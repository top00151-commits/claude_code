## ✅ 검증 ③ — sync 정상 동작 (3분)

### 01 검사기
1. `V3/01_검사기_완제품/실행_PMS(영업·PM).bat` 더블클릭
2. 정상 진행 확인:
   - [ ] 한글 깨짐 없음
   - [ ] "[1/2] sync.py 실행 중..." 진행
   - [ ] 로그에 "**부서입력 연동: 7개 부서**" 표시 (8개가 아님 ✓)
   - [ ] "[2/2] apply_standard.py 실행 중..." 진행
   - [ ] "완료! 5초 후 자동 종료"

3. 다시 PMS 파일 열어서 확인:
   - [ ] 데이터 행 정상 정렬
   - [ ] 6_관리코드발행대장 → 가공팀 컬럼 없음, 데이터 정상
   - [ ] 4_완료이력 → 가공팀 컬럼 없음, 데이터 정상
   - [ ] 활성 데이터 행 수 V2와 동일 (~106건)

### 02 자동화 — 동일 절차
- [ ] "**부서입력 연동: 6개 부서**" 표시 (7개가 아님 ✓)
