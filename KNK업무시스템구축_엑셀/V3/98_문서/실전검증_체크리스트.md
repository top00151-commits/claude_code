## ☑ 시나리오 — 신규 수주부터 납품완료까지 (5분 데모)

### Step 1 — 신규 수주 등록 (1분)

`01_검사기_완제품/KNK_검사기_완제품_PMS_2026.xlsx` 열기 → `1_프로젝트등록` 시트

빈 행에 다음 입력 (R5~R14 빈 공간 활용 또는 Ctrl+End 후 다음 행):
- 고객사: `삼성전자`
- 고객사담당자: `홍길동`
- 제품구분: `PBA` (드롭다운)
- 모델: `DEMO-MODEL-001`
- 품명: `검증용 데모 ICT`
- PO유형: `신규` (드롭다운)
- 영업단계: `수주확정` (드롭다운)
- 담당영업: `김영업`
- PM: `박PM`
- 수량: `1`
- 통화: `KRW` (드롭다운)
- 단가: `100000000` (1억)
- 수주일: `2026-05-01`
- 납기일: `2026-08-31`
- 진행상태: `진행중` (드롭다운)
- 부서 컬럼(C24~C31)에 담당자명 입력 (예: 설계팀=정민규, 검사기팀=홍길동, 가공팀은 빈 칸=미참여)

저장 후 닫기.

### Step 2 — sync 실행 (30초)

`01_검사기_완제품/실행_PMS(영업·PM).bat` **더블클릭**

(또는 명령 프롬프트:)
```cmd
cd "C:\Users\top00\JR\Claude 코드\KNK업무시스템구축_엑셀\V2\01_검사기_완제품"
python -X utf8 scripts\sync.py
python -X utf8 scripts\apply_standard.py
```

확인 사항:
- ☑ 한글 정상 표기 (chcp 65001 + BOM)
- ☑ "검사기_완제품 동기화 시작" 메시지
- ☑ "관리코드 채번: N건" — 신규 수주에 코드 발번
- ☑ "수주번호 생성: N건"
- ☑ "동기화 완료!" 표시 (백슬래시 없음)
- ☑ apply_standard 단계 정상 진행 (BEL 버그 패치 검증)
- ☑ "완료! 5초 후 자동 종료" 표시

### Step 3 — 결과 확인 (1분)

`KNK_검사기_완제품_PMS_2026.xlsx` 다시 열기 → `1_프로젝트등록` 시트:
- ☑ 새로 입력한 행에 **관리코드 자동 채워짐** (예: T1052605)
- ☑ **수주번호 자동 채워짐** (예: M-260501)
- ☑ **D-day 자동 계산됨** (납기일 2026-08-31 기준 음수 표시 `-122일`)
- ☑ **금액 1억** 표시 (수량 × 단가)

`6_관리코드발행대장` 시트:
- ☑ 새 관리코드 1건 추가됨

### Step 4 — 부서 진척 입력 (1분)

`KNK_검사기_완제품_설계팀_입력_2026.xlsx` 열기:
- ☑ 새 관리코드가 자동 표시됨 (`1_부서진척입력` 시트)
- 세부항목 1~4에 진척률 입력 (예: 0.5 = 50%)

- ☑ `진행률_매트릭스`: 새 프로젝트 행 추가
- ☑ `프로젝트목록`: 새 관리코드 행 추가
