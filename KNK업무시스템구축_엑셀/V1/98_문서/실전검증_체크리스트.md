# 실전 투입 전 검증 체크리스트

> **목적**: 대표이사 김정락이 직접 5분 데모로 전체 흐름 검증
> **전제**: 시뮬레이션 데이터(검사기 9건·자동화 7건·소모품 8건)가 이미 입력된 상태
> **기준일**: 2026-04-15 (V4 배포본)

---

## ☑ 시나리오 — 신규 수주부터 납품완료까지 (5분 데모)

### Step 1 — 신규 수주 등록 (1분)

`01_검사기_완제품/KNK_검사기_완제품_PMS_2026.xlsx` 열기 → `1_프로젝트등록` 시트

빈 행에 다음 입력:
- 고객사: `삼성전자` (드롭다운)
- 품명: `검증용 데모 ICT`
- PO유형: `신규` (드롭다운)
- 수주단계: `수주확정` (드롭다운)
- 수량: `1`
- 통화: `KRW` (드롭다운)
- 단가: `100000000` (1억)
- 수주일: `2026-04-15`
- 납기일: `2026-08-31`
- 진행상태: `진행중` (드롭다운)

저장.

### Step 2 — sync 실행 (30초)

명령 프롬프트 또는 PowerShell:
```cmd
cd "C:\Users\top00\JR\HAIST Innovation\매출흐름업무시스템_완성품_자재_소모품\HAIST_WORKS_baby\01_검사기_완제품"
python scripts/sync.py
```

확인 사항:
- ☑ "검사기_완제품 동기화 시작" 메시지
- ☑ "관리코드 채번: N건" — 신규 수주에 코드 발번
- ☑ "수주번호 생성: N건"
- ☑ 에러 없이 "동기화 완료" 표시

### Step 3 — 결과 확인 (1분)

`KNK_검사기_완제품_PMS_2026.xlsx` 다시 열기 → `1_프로젝트등록` 시트:
- ☑ 새로 입력한 행에 **관리코드 자동 채워짐** (예: T0102604)
- ☑ **수주번호 자동 채워짐** (예: KNK-T-2604-010)
- ☑ **D-day 자동 계산됨** (납기일 2026-08-31 기준)
- ☑ **금액 1억** 표시 (수량 × 단가)

`6_관리코드발행대장` 시트:
- ☑ 새 관리코드 1건 추가됨

### Step 4 — 부서 진척 입력 (1분)

`KNK_검사기_완제품_설계팀_입력_2026.xlsx` 열기:
- ☑ 새 관리코드(T0102604)가 자동 표시됨 (`1_부서진척입력` 시트)
- 세부항목 1~4에 진척률 입력 (예: 0.5 = 50%)
- 저장

다시 sync 실행:
```cmd
python scripts/sync.py
```

PMS 파일 다시 열기 → 새 프로젝트 진척률이 갱신됨 (☑ 0% → 약 6% 표시 = 50% × 1/8 부서)

### Step 5 — 대시보드 갱신 (30초)

```cmd
cd "..\10_대시보드"
python scripts/gen_dashboard.py
```

`KNK_경영현황_2026.xlsx` 열기:
- ☑ `매출총괄` 시트: 검사기 완제품 진행 8건 → **9건으로 증가**
- ☑ `매출총괄` 시트: KRW 합계 **1억 증가**
- ☑ `프로젝트목록` 시트: 새 관리코드 T0102604 행이 추가됨

### Step 6 — 납품완료 처리 (1분)

PMS 파일 → `1_프로젝트등록` 시트
- 새 프로젝트 진행상태를 `납품완료`로 변경
- 저장

sync 실행:
```cmd
cd ..\01_검사기_완제품
python scripts/sync.py
```

확인:
- ☑ `1_프로젝트등록`에서 새 프로젝트가 사라짐
- ☑ `4_완료이력`으로 자동 이동됨

대시보드 다시 갱신:
- ☑ 검사기 완제품 진행 -1, 완료 +1

## ☑ 모듈별 점검 항목

### 01/02 완제품
- ☑ PMS 파일 7시트 모두 열림
- ☑ 부서 입력 파일 8개(검사기) / 7개(자동화) 모두 열림
- ☑ 시트 보호 (비밀번호 `knk2026`) 적용 — 자동 셀 잠금 확인
- ☑ 드롭다운 작동 — 고객사·PO유형·통화·진행상태
- ☑ 헤더에 마우스 올리면 설명 팝업 (Comment) 표시
- ☑ 금액 컬럼 우측정렬 + ₩ 또는 $ 표시

### 05 소모품
- ☑ 출하대장 파일 2시트(소모품목록·출하관리) 열림
- ☑ 시뮬 데이터 8건 출하 표시
- ☑ 금액 자동 계산 (수량 × 단가)

### 10 대시보드
- ☑ 매출총괄·프로젝트목록 2시트 열림
- ☑ 검사기 9 + 자동화 7 + 소모품 8 = **24건** 집계
- ☑ R1·R2 좌측 정렬, 슬로건 표시
- ☑ 합계 행 정확히 계산됨
