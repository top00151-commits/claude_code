# 사용법 — 05 소모품

> **대상**: 구매·창고·총무 담당자
> **목적**: 소모성 자재 품목 등록 및 출하 기록 관리
> **기준일**: 2026-04-15 (V4 배포본)

---

## 1. 폴더 구성

```
05_소모품/
└── KNK_소모품_출하대장_2026.xlsx
    ├── [소모품목록]   ← 1. 품목 마스터 등록
    └── [출하관리]     ← 2. 일별 출하 기록
```

---

## 2. 입력 순서 (2단계)

### 단계 1 — 품목 등록 (소모품목록 시트)

신규 소모품이 들어오면 먼저 등록합니다.

| 입력 컬럼 | 예시 | 비고 |
|---|---|---|
| NO | 1, 2, 3... | 순번 |
| 품목코드 | CON-001, FIL-001 | 자체 코드 부여 |
| 품명 | 연결 케이블 2m | |
| 카테고리 | 케이블류 / 소모성자재 / 구동부품 / 유지관리 / 센서류 | |
| 단위 | EA / SET / KG / L / BOX | 드롭다운 |
| 단가 | 8500 | KRW |
| 거래처 | SDI, 한일필터, 미쓰보시 | |
| 보관위치 | 원자재창고A, 원자재창고B | |

---

### 단계 2 — 출하 기록 (출하관리 시트)

자재 출하 시마다 한 줄씩 기록.

| 입력 컬럼 | 자동 계산 |
|---|---|
| 관리코드·수주번호 | (선택) — 특정 프로젝트로 출하 시 |
| 품명·규격 | |
| 수량·단가 | **금액 = 수량 × 단가 (자동)** |
| 발주일·입고상태·입고일 | 드롭다운 (입고완료 / 입고대기) |
| 출하일·출하상태 | 출하완료 / 진행중 / 미출하 |
| INV번호·출하차수 | (해외 출하 시) |
| 거래유형·통관구분 | (해외 출하 시) |

## 5. 자주 묻는 질문

**Q. 출하했는데 대시보드에 안 보여요.**
A. 출하관리 시트의 `품명` 또는 `금액` 컬럼이 비어있습니다. 둘 다 채우면 대시보드에 자동 집계됩니다.

**Q. 같은 품목을 여러 번 출하하면 매번 등록해야 하나요?**
A. 네. 출하관리 시트는 출하 이력을 누적합니다. 품목 등록(소모품목록)은 한 번만 하면 됩니다.

**Q. 향후 자재마스터 ERP가 도입되면?**
A. 현재 소모품 모듈은 그대로 유지하면서, 자재마스터에서 가져오는 구조로 발전 예정입니다.
