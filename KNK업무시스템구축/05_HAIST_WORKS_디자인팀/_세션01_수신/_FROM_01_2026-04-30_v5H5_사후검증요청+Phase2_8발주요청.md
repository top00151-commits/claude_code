### 검증 1. 페이지별 옛 sage 잔존
```bash
# 124 템플릿 각각에서 var(--sage-*) 직접 사용 측정
for tpl in app/templates/*.html; do
  n=$(grep -cE 'var\(--sage-[0-9]+\)' "$tpl")
  if [ "$n" -gt 0 ]; then echo "$tpl: $n"; fi
done
# 빅터(05) 평가: Phase 2 분리 시 우선 정리할 페이지 식별
```

### 검증 2. 인라인 #hex 잔존
```bash
grep -rE 'style="[^"]*#[0-9a-fA-F]{3,6}' app/templates/ | wc -l
# 빅터(01) Phase 1 미처리 / Phase 2 의존
```

### 검증 3. Standalone 페이지 별도
```bash

```
