## 1. 현재 상태 (개선 전)

| 항목 | 현재 | 문제 |
|---|---|---|
| 페이지 헤더 | 단순 h1 + muted 안내 | breadcrumb·lock-badge·sage 액센트 부재 |
| 본문 | 체크박스 표 1개 (이름·직급·5권한) | 맥락·통계·이력 0건 |
| 시드 권한자 표시 | `<span class="tag">임원</span>` 라벨만 | "자동 권한·해제 불가" 시각 분리 약함 |
| 다른 팀 이동 (CEO/임원) | 표 하단 작은 버튼 그룹 | 발견성 낮음 |
| 변경 이력 | 미노출 | notifications 자동 기록되지만 화면에 없음 |
| 모듈 설명 | 컬럼 헤더 title 속성만 | 호버해야 보임 |
| 톤 | KNK 레드 그라디언트(구) | 힐링 sage 미적용 |

## 2. 요청 사양 (필수)

### 2-1. 페이지 영역 5단

```
┌──────────────────────────────────────┐
│ ① page-head (breadcrumb + accent + 🔒 팀장+ 뱃지)  │
├──────────────────────────────────────┤
│ ② 권한 부여 현황 KPI (매출·자재·변경·티켓 카운트)   │
├──────────────────────────────────────┤
│ ③ 메인 — 권한 토글 표 (현재 팀)                    │
│   · 시드 권한자(CEO/임원) 음영 + "자동" 라벨        │
│   · 일반 멤버 토글 가능                            │
│   · 헤더 ⓘ 호버 → 모듈 상세 설명 카드             │
├──────────────────────────────────────┤
│ ④ 좌: 최근 권한 변경 7일 (notifications 미니 타임라인) │
│   우: 다른 팀 빠른 이동 칩 그리드 (CEO/임원만 노출)  │
├──────────────────────────────────────┤
│ ⑤ 저장 바 (sticky bottom · 변경 카운트 + 저장)     │
└──────────────────────────────────────┘
```

### 2-2. 색·컴포넌트 (시안 12B 토큰 그대로)
- 헤더: `--grad-sage` accent · sage-50 배경
- KPI: 시안 #20 의 `.sum-card` 4개 (sum-ico.users / .active / .new / .lock)
- 권한 표: 시안 #20 의 `.user-tbl` (sage-50 thead, perm-chips 사용 가능)
- 토글: 시안 #20 의 `.role-pills` 패턴 활용 (체크박스 → 알약 토글)
- 시드자 행: `.dept-stat` 톤 (sage-50 배경 + 자물쇠 아이콘)
- 타임라인: 시안 #20 의 `.audit-list` 그대로
- 팀 칩: `.role-pill` 패턴 (현재 팀 active)
- 저장 바: `--grad-brand` (KNK 레드 CTA)

### 2-3. 데이터 변수 (백엔드 1차 보강 후 제공)

| Jinja 변수 | 형태 | 설명 |
|---|---|---|
| `team` | dict | 현재 팀 (id, name) |
| `members` | list[dict] | 팀원 (id, name, rank, role, can_use_sales, can_use_logistics, can_edit_changes, can_close_tickets, is_admin) |
| `kpi` | dict | `{sales: N, logi: N, chg: N, tkt: N, total: N, seed: N}` 권한별 부여 인원 |
| `recent_changes` | list[dict] | `[{when, actor, body}]` 최근 7일 권한 변경 (notifications 에서 추출) |
| `all_teams` | list[dict] | CEO/임원만 — 전 팀 목록 (id, name, member_count) |
| `is_admin_actor` | bool | CEO/admin 여부 (is_admin 토글 표시 분기) |
