### 1. 기존 Victor 모달 완전 제거

- `base.html` 상단 `#victorBox` 트리거 (topbar 버튼) **제거**

- 잔존 참조 0건 (`grep -c "victorModal|victorBox|victorOpen|renderVictorResult" = 0`)

### 2. 사이드 도크 신규 구현 (base.html + base_logi.html)

- `.dock-tab` (우측 세로 탭, 항시 노출, ⌘K 힌트 포함)
- `aside.dock#victorDock` (420px 패널)
  - 헤더: 아바타 + 상태 pulse dot · 초기화 · 접기 버튼
  - 맥락 칩 (서버에서 Jinja 주입): 현재 페이지별 3개 추천
  - 대화 영역: 타이핑 인디케이터 + 말풍선 (user / ai 좌우 구분)
  - 입력: textarea 자동 grow + quick chips 5개 + 전송 버튼

### 3. 상태 유지

- `localStorage['haist_victor_dock_open']` 으로 열림 상태 페이지 이동 간 복원
- `body.dock-open` 클래스 토글로 `.main.has-sidebar` 및 `.layout-wrap` **margin-right: 420px** 자동 적용
- topbar 도 `padding-right: 420px` 로 함께 밀림 (overlap 방지)

### 4. 백엔드 변경 — 맥락 칩

`main.py` 에 `VICTOR_CONTEXT_CHIPS` 21 route 매핑 + `_victor_chips_for_path()` 헬퍼 추가:
```
/home, /daily, /calendar, /notifications, /feed, /now,
/progress, /tickets, /issues, /changes, /dashboard, /bottlenecks,
/admin, /sales, /logistics, /parts, /po, /stock/movements,
/rates, /board/company, /board/team
```
- prefix match 로 `/changes/123` → `/changes` 칩 재사용
- `ctx()` helper에 `victor_chips` 자동 주입 → 모든 템플릿에서 즉시 사용 가능

### 5. "Do" 스타일 실행 버튼

- `victor.py::_link(style=...)` 이미 `"do"` 지원 확인 (변경 불필요)
- 프런트 핸들러: `data-do="1"` 링크 클릭 시 `confirm()` → `fetch(POST, {}, '{}')` → 결과 alert
- 나머지 링크는 기본 이동 (도크 열림 상태는 localStorage 로 유지)

### 6. 접근성 · 단축키

- `Ctrl+K` / `Cmd+K`: 도크 토글
- `Esc`: 접기 (입력창 비우기 우선 → 한 번 더 Esc 접기)
- `Enter` 전송 · `Shift+Enter` 줄바꿈
- `prefers-reduced-motion`: 제안 #07 에서 이미 전역 추가 → 도크 애니메이션도 자동 비활성화
