## 2. 토큰 정의 — 6단계 (`--text-xs` ~ `--text-2xl`)

`00_design_tokens.css` `:root` 절에 다음 6개 추가.

```css
:root {
  /* font scale tokens — 한국어 가독성 본문 16px(md) 기준 */
  --text-xs:  clamp(11px, 0.75rem  + 0.20vw, 13px);   /* 캡션·메타 */
  --text-sm:  clamp(13px, 0.875rem + 0.20vw, 15px);   /* 사이드바·탭·버튼 보조 */
  --text-md:  clamp(15px, 1.000rem + 0.25vw, 17px);   /* 본문 (한국어 16px 권장) */
  --text-lg:  clamp(17px, 1.125rem + 0.30vw, 20px);   /* 카드 제목·KPI 라벨 */
  --text-xl:  clamp(20px, 1.375rem + 0.40vw, 24px);   /* 섹션 헤더 */
  --text-2xl: clamp(24px, 1.625rem + 0.50vw, 30px);   /* 페이지 타이틀·KPI 수치 */
}
```

- `clamp(min, base, max)` 3인자 형식 강제 — base는 `rem + vw` 합산형으로 유체 타이포 보장.
- 최소값(min)은 모바일(320px) 가독성 하한. 최대값(max)은 와이드(1920px+) 과대 방지.

---

## 3. 한국어 가독성 — 본문 16px (md) 권장

- `--text-md`(=1rem 기준 = 16px) → 본문 기본. 한국어 글자 폭이 영문보다 넓어 14px 미만은 가독성 저하.
- `body { font-size: var(--text-md); }` 1회 선언으로 전역 적용.
- `<p>`, `<li>`, 일반 텍스트는 별도 폰트 지정 금지 — body 상속.

---

## 4. 메뉴 ≤ 본문 비례 보장 룰 (역전 방지)

04 권고 반영: "메뉴 폰트 ≤ 본문 폰트" 강제.

| UI 영역 | 토큰 | 비고 |
|---|---|---|
| 캡션·시간·메타 | `--text-xs` | 보조 정보 |
| **사이드바 메뉴** | **`--text-sm`** | 본문보다 1단계 작게 (인라인 17px → 토큰 치환) |
| **본문 (`body`, p, li)** | **`--text-md`** | 기준 (한국어 16px 권장) |
| 카드 헤더·KPI 라벨 | `--text-lg` | 본문보다 1단계 크게 |
| 섹션 타이틀 | `--text-xl` | — |
| 페이지 타이틀·KPI 수치 | `--text-2xl` | — |

**위반 사례 차단**: 사이드바 17px(=lg 영역) > 본문 14px(=sm 영역) 같은 역전 금지. 자기검증 grep §6 참조.

## 6. 자기 검증 grep 패턴

```
# G17-1: 토큰 6종 정의 확인
grep -cE "^\s*--text-(xs|sm|md|lg|xl|2xl)\s*:" 00_design_tokens.css
# 기대: 6

# G17-2: clamp() 사용 확인
grep -cE "clamp\([^)]+\)" 00_design_tokens.css
# 기대: 6 이상 (각 토큰 1회 이상)

# G17-3: 사이드바 인라인 변수 폐지 추적
grep -nE "--sb-font-size|--sb-head-size" style.css
# 기대 (전환 후): 0건 — 1건 이상이면 토큰 미적용

# G17-4: 본문 폰트 토큰 사용
grep -nE "body\s*\{[^}]*font-size\s*:\s*var\(--text-md\)" style.css
# 기대: 1건

# G17-5: 메뉴 ≤ 본문 비례 점검 (역전 차단)
grep -nE "(side|nav|menu)[^{]*\{[^}]*font-size\s*:\s*var\(--text-(lg|xl|2xl)\)" style.css
# 기대: 0건 (메뉴가 본문보다 크면 위반)
```

검증 대상: `01_HAIST_WORKS/static/00_design_tokens.css` + `static/style.css` + `05_HAIST_WORKS_디자인팀/03_시안/*`
