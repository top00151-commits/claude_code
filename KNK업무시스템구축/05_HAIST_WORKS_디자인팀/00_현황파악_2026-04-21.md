## 어려움 후보 Top 5

### 1. /feed (부서 피드) — 필터 칩 9개 + 뷰 탭 3개, 시각 부하 최대
- 필터 칩 9개 (전체/지연/진행중/완료/대기/댓글있음/리스크/멘션/리셋)
- 보기 탭 3개 (리스트/칸반/캘린더)
- 동시 정보 4계층: 팀장 요약 grid → 필터 bar → 보기 탭 → 팀 카드 리스트
- 알림 밴드 "🔴지연 5 | ⚠️리스크 3 | ❓멘션 2" 병행 → 강조 과밀
- 상태 태그 4색(황/녹/적/회) + 배지 4개 동시 → 색상 학습 가파름

**원칙 차원 제안**: 필터 칩 "긴급 그룹 / 상태 그룹" 분리, 보기 탭 기본=리스트 고정, 카드 배지 중요도순 정렬, 팀별 미세 배경색 구분.

### 2. /progress (진행률 매트릭스) — 프로젝트 × 12공정 + 필터 + 범례 + 모달
- 필터: 사업부(드롭) + 고객사(텍스트) + 상태(드롭) + 필터/초기화 버튼 → 6 CTA
- 색상 5가지 (예정:회/진행:황/완료:녹/지연:적/미설정:흐림) 신호등 해석 요구
- 매트릭스 `min-width 1400px` → 가로 스크롤 필수
- 첫 사용 모달 자동 인터럽트 1회

**원칙 차원 제안**: 필터를 "기본/고급" 아코디언 분리, 범례를 매트릭스 옆 sticky, 셀 상태 변경 확인 모달 또는 우클릭 메뉴, 첫 사용 모달을 도움 버튼으로 전환.

### 3. /dashboard (전사 대시보드) — KPI 6 + 내러티브 + 14팀 테이블 + CTA 20+
- 테이블 12열 (코드·팀명·팀장·인원·오늘작성·참여율·주간카드·완료·진행·지연·공수·바로가기)
- 내러티브 카드 2-3줄에 6개 정보 압축 (팀명·담당자·직급·제목·프로젝트·다음계획)
- KPI "오늘 참여율" 클릭 → `/admin/reminders` 맥락 점프

**원칙 차원 제안**: 내러티브 최대 2줄 제한 또는 지연만 강조, 참여율 열을 비율바만 단순화, KPI 클릭 시 동일페이지 필터 토글 팝오버.

### 4. /team (팀 현황) — 팀원 수만큼 섹션 반복, 세로 스크롤 최장
- 요약카드 7개 1행 복단 배치
- 팀원별 섹션 6~15개 × 각 섹션 헤더(이름/직급/통계5개) 반복
- 메뉴 depth 5단계 (제목→팀선택→날짜→요약→팀원섹션→카드상세)

**원칙 차원 제안**: 날짜 네비 상단 sticky, 팀원 섹션 아코디언 기본닫힘, 요약카드 2행(4+3) 또는 3행 재배치, 카드 배지 중요도 정렬.

### 5. /home (업무현황) — KPI + 팀카드 + 통계 + 이월 리스트 4계층
- KPI 카드 스파크라인(7개 막대) → 시간 경향 해석 요구
- 팀 신호등 + 이모지 + 텍스트 병행 → 중복 표현
- 빠른입력 포커스 시 확장 (display:none → flex) → UX 예측 어려움
- 맨아래 물류통계 추가 → 스크롤 끝 길어짐

**원칙 차원 제안**: 스파크라인 호버 시만, 팀 신호등 색상 단독, 빠른입력 항상 펼침 또는 하단 sticky, 물류통계 토글 기본숨김.
