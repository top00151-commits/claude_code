### [DSG-V01] 사이드바 "사용 가이드" 메뉴 2회 중복
- 정성진 사이드바 첫 노출 시 📚·📖 두 개 라인이 동일 라벨로 노출
- 사용자 혼란 즉시 발생 — 어느 걸 눌러야 하나?

### [DSG-V02] 진행률 대시보드 도달성 0
- 사이드바 메뉴 없음 + 검색 0결과 = 직접 URL만 가능
- 설문 §3 매일 아침 1순위 요구 메뉴인데 **메뉴 등재 누락**
- 05가 사이드바 IA 재설계 시 "내 업무" 그룹 또는 "전사" 그룹 상단에 우선 배치 권장

### [DSG-V03] 일일카드 placeholder 페르소나 적합성
- "예) 갤럭시 S27 ICT 1차 셋업 점검" — 검사기/영업 예시
- 구매·관리·라이프밸류 팀이 보면 "내 업무 아닌데" 헷갈림
- **팀별 sector 기반 placeholder 차별화**: 구매팀이면 "예) KMC 5월 견적 회신 검토" 등

### [DSG-V04] "📌 다음 계획 (내일 할 일)" 필수 안내 부재
- `*` 표기만 있고 왜 필수인지 안내 0
- 05가 "이 항목은 내일 카드 자동 carry-forward 에 사용됩니다" 같은 헬프 텍스트 정의 권장

### [DSG-V05] KPI 표기 일관성
- 일일업무 페이지 이번 주 KPI: "1건 / 0완료 / 1진행중 / 0지연 / 2.0h 누적"
- 팀 페이지 KPI: "7 팀 인원 / 0/7 오늘 작성 / 0/0 주간 완료"
- **분수 표기 vs 단순 숫자** 혼재. 통일 가이드 필요.

### [DSG-V06] 진행률 매트릭스 컬럼 라벨 잘림
- 데스크탑 1568px 폭에서도 "기술영"·"설계팀" 라벨이 4글자에 잘림
- 좁은 셀에 긴 팀명 표시 — 약어 표준 또는 hover 툴팁 필요

## 의뢰서 §S1-B (빅터 도크 포지셔닝) 보강 검증

### 이전 판정 (2026-04-25)
> "🔴 FAIL — `.dock-tab` 1건 잔존, 2차 수정 미반영"

### 시각 검증 결과 (2026-04-27)

CEO 홈 + 정성진 홈 둘 다:
- 빅터 도크가 우측 패널로 잘 자리잡음 (사이드 도크)
- 헤더의 사용자 칩(김정락 대표이사·정성진 매니저)·로그아웃 버튼·언어 셀렉터·알림 종 모두 **가려지지 않음**
- 빅터 도크 **다른 페이지로 이동해도 우측에 유지** (페이지 컨텍스트에 따라 빠른 질문 자동 변경)
- 메인 컨텐츠와 도크 사이 시각 분리 명확

### 결론

- `.dock-tab` 마크업 자체는 잔존하나 **시각적 가림·버튼 겹침은 발생하지 않음**.
- HTML 단순 검사로는 "잔존 = FAIL" 이지만 실제 사용자 경험은 **PASS** 수준.
- 마크업 정리는 코드 위생 차원에서 처리 권장 (S3 수준), 시각 차단은 없음 — **S1 → S3 으로 다운그레이드 제안**.
