## 참고 — 구현 힌트 (권장, 강제 아님)

- 탭 스위칭은 `?tab=my|team|all` 쿼리스트링 기반 서버 렌더. SPA 불필요.
- 탭 기본값:
  ```python
  default_tab = 'my'
  if user.role in ('leader',): default_tab = 'team'
  if user.role in ('ceo','admin','executive'): default_tab = 'all'
  ```
- 이미 존재하는 `team_data`, `hw_counts`, `narratives` 등의 컨텍스트는 그대로 재사용. Jinja `{% if tab == 'my' %}` 블록 분기만 추가.
