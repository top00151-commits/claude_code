## TL;DR

기존 전체화면 모달을 **사이드 도크** 로 교체. 대표 지시 4가지 요구 모두 충족:

1. **"항상 오른쪽 옆에"** → 우측 세로 탭 항시 노출 (🤖 빅터 ⌘K)
2. **"슬라이드처럼 열었다 접고"** → 420px 패널 슬라이드 + `body.dock-open` 토글
3. **"계속 보이게"** → localStorage 로 열림 상태 유지, 페이지 이동해도 복원
4. **"바로 옆에서 실행"** → 답변 카드에 "바로 실행" (초록 do 버튼) 1클릭 액션

백엔드 (`/api/victor/ask` + `victor.py`) **그대로 재사용 가능**. 프런트 UX만 교체.

## 백엔드 확장 (권장, 필수 아님)

`victor.py` 의 `_link()` 시그니처에 `style` 옵션으로 **"do"** 추가:
```python
def _link(label, href, style="primary"):
    """style: primary | secondary | do (바로 실행)"""
    return {"label": label, "href": href, "style": style}
```

실행 버튼 예:
- `"결재 올리기" href="https://office.hiworks.com/..." style="primary"` (외부 이동)
- `"전체 확인 처리" href="#" style="do"` — 프런트에서 `onclick` 가로채 `/api/changes/mark_all_read` POST

## 리스크·제약 (간단)

- 기존 모달 사용자 재학습 1회 필요
- 1024~1280px 중간 해상도에서 도크 열면 메인 영역 860px — 테이블 스크롤 가능성. 사용자가 접으면 해소.
- "바로 실행" 액션은 확인 토스트 권장 (실수 방지).
- `prefers-reduced-motion` 미디어쿼리로 애니메이션 비활성화 권장.
