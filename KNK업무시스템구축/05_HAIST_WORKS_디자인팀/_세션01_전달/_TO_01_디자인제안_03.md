## TL;DR

`/feed` 는 alert-band 3칩 + filter chip 9개 = **"지연·리스크·멘션" 이 두 군데 중복 노출**. 5가지 변경 부탁드립니다.

1. **alert-band 3칩 제거** → "지연 N · 리스크 N · 멘션 N" 요약 라인만 (line 34–48)
2. **filter chip 9→4 + 더보기 5** (line 56–67)
3. **dense-list 팀 collapse 규칙 통일**: "내 팀 펼침" 1규칙 + 접힌 팀에 긴급 N 뱃지 (line 87–91)
4. **뷰 탭 리스트 강조**: 칸반·캘린더를 "다른 방식으로 보기 ▾" 토글로 축약 (line 50–54)
5. **filter-bar `position: sticky`** — 긴 리스트 스크롤 중에도 필터 유지

## 구현 힌트 (권장)

### alert-band 요약 (Jinja 만 수정)
```html
<div class="alert-band">
  <span class="ab-title">{{ user.name }} {{ user.rank or '' }}님, 지금 봐야 할 것</span>
  {% if all_delay==0 and all_risk==0 and all_mentioned==0 %}
    <span class="ab-allgreen">✅ 긴급건 없음</span>
  {% else %}
    <span>지연 <b>{{ all_delay }}</b> · 리스크 <b>{{ all_risk }}</b> · 내게 멘션 <b>{{ all_mentioned }}</b></span>
  {% endif %}
</div>
```

### filter chip 축약
- line 59–66 중 4개(`전체·지연·진행중·완료`)는 상시 노출
- 나머지 5개(`대기·댓글·리스크·멘션·초기화`)는 `<div class="chips-more hidden">` 에 넣고 "더보기" 버튼으로 토글
- CSS: `.chips-more.hidden { display: none; }`

### 팀 collapse 통일 (Jinja)
```diff
- {% if user.role in ['ceo','executive','admin'] %}
-   {% set default_open = data.has_urgent %}
- {% else %}
-   {% set default_open = data.has_urgent or is_my_team %}
- {% endif %}
+ {% set default_open = is_my_team %}
```
팀 헤드에 긴급 뱃지 추가:
```html
{% if data.has_urgent %}
  <span class="dl-urgent-badge">긴급 {{ data.urgent_count }}</span>
{% endif %}
```
