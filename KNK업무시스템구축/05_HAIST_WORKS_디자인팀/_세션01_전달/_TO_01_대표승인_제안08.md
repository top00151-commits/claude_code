## 백엔드 확장 (선택 · 권장)

`app/victor.py`:
```python
def _link(label: str, href: str, style: str = "primary") -> dict:
    """style: primary (빨강·이동) | secondary (회색·관련) | do (초록·즉시 실행)"""
    return {"label": label, "href": href, "style": style}
```

"do" 스타일 액션 예:
- "전체 확인 처리" → `POST /api/changes/mark_all_read`
- "담당자에게 알림" → `POST /api/notify`
- "Excel 다운로드" → `GET /admin/export?type=...`

프런트에서 `onclick` 가로채고 fetch 호출 + 성공 토스트.
