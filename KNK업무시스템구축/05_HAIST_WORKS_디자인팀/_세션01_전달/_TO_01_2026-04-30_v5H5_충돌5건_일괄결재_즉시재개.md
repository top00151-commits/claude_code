## ⑤ Pretendard·로고 — 사내 호스팅 + 자동 변환

### Pretendard
**외부 CDN 사용 금지** — KNK 라이선스 정책 (`_TEAM_ORIENTATION.md` 2026-04-21) **외부자산 0건 원칙**. Pretendard는 SIL OFL 1.1 자유 라이선스 → **사내 정적 호스팅** 안전.

### 처리
1. `app/static/fonts/pretendard/` 폴더 신설
2. https://github.com/orioncactus/pretendard `dist/web/static/` 일괄 다운로드
3. `app/static/css/v5_tokens.css` 상단에 `@font-face` 추가:
   ```css
   @font-face {
     font-family: 'Pretendard';
     src: url('/static/fonts/pretendard/Pretendard-Regular.woff2') format('woff2');
     font-weight: 400;
     font-display: swap;
   }
   /* 200~900 9 weight 모두 추가 */
   ```
4. 시안의 `<link href="https://cdn.jsdelivr.net/...">` **삭제** + v5_tokens.css 자동 적용

→ **외부 의존 0건 / 시안 시각 100% 동일**.

### 로고
시안 `<img src="../../../../../01_HAIST_WORKS/static/logo.png">` (시안 폴더 상대경로) → **`<img src="/static/logo.png">` (활성 절대경로)** 자동 변환. 빅터(01) 권고 채택.
