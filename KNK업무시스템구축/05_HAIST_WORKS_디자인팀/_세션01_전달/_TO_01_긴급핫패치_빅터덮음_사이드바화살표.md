❌ **금지 패턴 1** — `app/templates/base.html`
```html
<div class="sb-toggle" onclick="toggleSidebar()">◀</div>
```

❌ **금지 패턴 2** — `static/style.css`
```css
.dock { position: fixed; top: 0; ... }   /* top: 0 절대 금지 */
body.dock-open .topbar { padding-right: 420px; }   /* topbar 시프트 절대 금지 */
```

**v2 통째 이식 시 권장 (시안 12B 와 정합)**:

```css
/* 옵션 A — 본 핫패치 유지 (position: fixed 유지, top 만 헤더 아래로) */

/* 옵션 B — 시안 12B 와 완전 동일 (.layout flex 자식으로 DOM 재배치) */
/* 시안 12B line 198 참조: */
.dock { width: var(--dock-w, 360px); flex-shrink: 0; ... }   /* position 없음 */
/* + .layout flex 컨테이너 안에 .sidebar / .main / .dock 3 칼럼 배치 */
```

→ **옵션 B가 시안 100% 정합** 이지만 DOM 재배치 비용 큼. 시간 부족하면 **옵션 A (본 핫패치 그대로)** 유지하면 헤더 안 가리는 요건은 충족.
