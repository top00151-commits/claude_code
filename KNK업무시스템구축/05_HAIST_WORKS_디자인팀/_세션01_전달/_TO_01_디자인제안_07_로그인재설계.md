## 구현 힌트

### 1. 로그인 재설계

`login.html` 을 완전히 대체. 시안 `06_login_브랜드스플릿.html` 의 HTML·CSS 를 그대로 차용 가능. 변경 포인트:
- `login.html` body block 전체 교체
- `style.css:232~245` + `1584~1620` 의 기존 `.login-*` 클래스는 **제거**
- 새 클래스: `.split`, `.brand-panel`, `.form-panel`, `.field`, `.btn-login`, `.lang-sel`, `.promises`
- CSS 캐시 키 갱신: `?v=20260424login`
- **힌트 블록 제거** (대표이사 ID 노출 해결)

### 2. 유동 타이포

`00_design_tokens.css` (05 폴더) 에 이미 추가 완료된 `--fs-fluid-*` 7토큰 → **01 의 `static/style.css:root` 에도 복사**:

```css
:root {
  --fs-fluid-xs:  clamp(10px, 0.55vw + 7px, 13px);
  --fs-fluid-sm:  clamp(12px, 0.60vw + 9px, 15px);
  --fs-fluid-md:  clamp(13px, 0.70vw + 10px, 16px);
  --fs-fluid-lg:  clamp(15px, 0.85vw + 11px, 19px);
  --fs-fluid-xl:  clamp(18px, 1.25vw + 13px, 26px);
  --fs-fluid-2xl: clamp(24px, 2.00vw + 16px, 40px);
  --fs-fluid-hero: clamp(32px, 4.50vw + 20px, 72px);
}
```

치환 대상 (우선순위):
- `.page-head h1` · `h2` → `var(--fs-fluid-xl)` / `var(--fs-fluid-lg)`
- `.hm2-kpi-val` · `.kpi-num` → `var(--fs-fluid-2xl)`
- `body` · 기본 → `var(--fs-fluid-md)`
- `.login-title` → `var(--fs-fluid-xl)`

**기존 `--fs-xs ... --fs-2xl` 은 삭제하지 말고 유지** (하위 호환).

### 3. 시각 차별화 6원칙

각 항목 `style.css` 수정만으로 가능:

| 원칙 | 핵심 CSS |
|---|---|
| KPI 카드 액센트 바 | `.hm2-kpi::before { content:''; position:absolute; top:0; left:0; right:0; height:3px; background: linear-gradient(135deg,#A5282C,#8B1E22); }` |
| KPI 호버 lift | `.hm2-kpi:hover { transform: translateY(-4px); box-shadow: 0 12px 28px rgba(15,23,42,0.08); }` |
| 버튼 호버 lift | `.btn-primary:hover { transform: translateY(-2px); box-shadow: 0 8px 20px rgba(165,40,44,0.35); }` |
| 상태 칩 도트 pulse | 시안 `07_visual_language.html` `.chip-new` 블록 차용 |
| 매트릭스 셀 hover scale | `.cell:hover { transform: scale(1.08); z-index: 2; }` |
| 플로팅 라벨 | 시안 `.field` 블록 차용 + input `placeholder=" "` (공백 1자) 필수 |
