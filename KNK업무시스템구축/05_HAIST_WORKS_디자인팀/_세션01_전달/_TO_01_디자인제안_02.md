## TL;DR

`/progress` 매트릭스는 `min-width:1400px` 로 **모바일에서 실질 사용 불가**. 현장·출장·해외 직원이 프로젝트 공정을 확인하지 못함. 5가지 변경 부탁드립니다.

1. **뷰 자동 전환**: 데스크톱 매트릭스 / 모바일 프로젝트 카드
2. **셀 단순화**: 색 5→3, 팀 약자 제거, 지연은 배지 위첨자
3. **첫 사용 모달 → 인라인 코치 배너** (전체화면 가림 해제)
4. **필터 2단 분리**: 기본 2요소 + [고급 필터 ▾] 토글
5. **JetBrains Mono 하드코딩 → `var(--font-mono)` 토큰화** (line 114, 140) — 정책 O-2 동반 적용

시안에서 실제 동작 확인 가능합니다.

## 구현 힌트 (권장, 강제 아님)

### 뷰 자동 전환 (CSS 만)
```css
@media (max-width: 1023px) {
  .matrix-table-wrap { display: none; }
  .card-list { display: block; }
}
@media (min-width: 1024px) {
  .matrix-table-wrap { display: block; }
  .card-list { display: none; }
}
```
같은 Jinja 컨텍스트를 매트릭스와 카드 양쪽에 렌더. JS 불필요.

### 셀 단순화
`progress_matrix.html` line 126~134 의 `if status == ...` 분기를 3 단계로 축약:
- `예정·대기` → `plan` (회색)
- `진행` → `prog` (연노랑 + 노랑테)
- `완료` → `done` (연초록 + 초록테)
- `지연` 은 **배경색이 아닌 배지 위첨자** (`position:absolute; top:-3px; right:-3px` 원형 빨간 점)
- `보류` 는 `plan` 으로 흡수하되 아이콘 "⏸" 추가 (선택)

### 첫 사용 모달 → 코치 배너
line 5~61 의 모달·스크립트 **전체 제거** 또는 아래로 축소:
```html
<div class="coach-bar" id="coachBar">
  💡 행 클릭=상세, 셀 색=상태, 빨간 점=지연 [닫기]
</div>
```
"? 안내" 버튼은 page-head 에 그대로 유지.

### 폰트 토큰화 (O-2)
line 114:
```html
<!-- before -->
<td style="... font-family:'JetBrains Mono',monospace; ...">
<!-- after -->
<td style="... font-family:var(--font-mono); ...">
```
line 140 동일. `--font-mono` 는 Pretendard 핸드오프의 `fonts_font-face_snippet.css` 에서 정의됨.
