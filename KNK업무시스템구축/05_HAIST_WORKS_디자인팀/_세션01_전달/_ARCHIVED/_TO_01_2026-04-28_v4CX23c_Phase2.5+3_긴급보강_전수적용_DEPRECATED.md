## 3. 즉시 처리 — 알리아스 레이어 (15분 작업, 전체 즉시 변환)

`static/style.css` 의 CX23c 토큰 정의 직후에 **다음 alias 블록 추가** — 모든 sage 변수가 CX23c로 자동 매핑:

```css
/* ============================================================
   SAGE → CX23c ALIAS LAYER (대표 직접 지시 2026-04-28)
   기존 페이지 125개의 var(--sage-*) 참조를 CX23c 톤으로 자동 변환.
   125개 템플릿 일일이 수정 안 하고도 즉시 시각 전환.
   ============================================================ */
:root {
  --sage-50:  var(--paper);          /* #FAFAF7 */
  --sage-100: var(--paper-2);        /* #F0EDE3 */
  --sage-200: var(--line-soft);      /* #D8D4CC */
  --sage-300: var(--line-soft);      /* #D8D4CC */
  --sage-400: var(--ink-4);          /* rgba(10,10,10,.55) */
  --sage-500: var(--ink-3);          /* rgba(10,10,10,.70) */
  --sage-600: var(--ink-2);          /* #2A2A26 */
  --sage-700: var(--ink);            /* #0A0A0A */
  --sage-800: var(--ink);            /* #0A0A0A */

  --grad-sage:   linear-gradient(135deg, var(--ink-2), var(--ink));
  --grad-header: var(--paper);       /* 헤더는 CX23c topbar가 별도 정의 */

  /* 기존 rose/accent 별칭 — CX23c 그대로 KNK 빨강 */
  --rose:        var(--accent);
  --rose-dk:     var(--accent-dk);
  --rose-soft:   var(--accent-soft);

  /* amber/peach/teal — 의미별 매핑 (KPI 그래프용) */
  --amber:       var(--ink-3);       /* 경고 → 약한 검정 (CX23c는 색 안 씀) */
  --peach:       var(--accent);      /* 부드러운 경고 → 빨강 */
  --teal:        var(--ink-3);       /* 정보 → 약한 검정 */
}
```

→ 적용 즉시 사이드바·KPI 카드·버튼·테이블 등 **모든 sage 변수 참조가 CX23c 톤으로 자동 변환**.

⚠️ **단**, 인라인 색상 (`style="background:#F5E4E2"` 등)은 alias로 안 잡힘 — §4 페이지별 정밀 정리 필요.
