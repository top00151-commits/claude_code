## TL;DR

`/admin` 8 섹션 중 4개가 테이블 → 스크롤 과도 + 위험기능 섞임. 5가지 변경 부탁드립니다.

1. **탭 3분할**: 대시보드 / 데이터 / 보안
2. **4 테이블 → 요약 카드 + "목록 관리 →" 링크** (테이블은 `/admin/users`·`/admin/projects`·`/admin/customers` 로 이동)
3. **그라디언트 카드 2개 → 단색 카드** 통일 (line 20–29)
4. **비번 재생성 2단계 확인 모달** — 체크박스 + 대표이사명 입력 (04 OPS-008 연계)
5. **사용자 테이블 "물류" 컬럼 → "자재 권한"** 이름 변경 + 규칙 명시 (04 OPS-001 연계)
