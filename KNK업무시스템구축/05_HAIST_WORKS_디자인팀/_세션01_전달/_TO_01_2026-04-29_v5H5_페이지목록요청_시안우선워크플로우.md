## 4. 요청 사항 — 페이지 목록 응답서

### 4-A. 응답 양식

마크다운 표로 응답 부탁드립니다:

```markdown
| # | 카테고리 | 라우트 (URL) | 템플릿 파일 (경로) | base.html 상속 | 우선순위 P0/P1/P2 | 권한 | 비고 |
|---|---|---|---|---|---|---|---|
| 1 | 대시보드 | `/` | `templates/dashboard.html` | ✓ | P0 | 전체 | 통합 메인 |
| 2 | 매트릭스 | `/sales/dashboard` | `templates/sales/dashboard.html` | ✓ | P0 | 영업 | 매출 대시 요약 |
| 3 | standalone | `/login` | `templates/auth/login.html` | ✗ | P0 | 비로그인 | 로그인 페이지 |
| ... | ... | ... | ... | ... | ... | ... | ... |
```

다음 카테고리로 분류해주세요:

### 4-C. 추가 정보 요청

각 페이지에 대해 다음 정보 가능하면 포함:

1. **base.html 상속 여부** — `{% extends "base.html" %}` 있음/없음
2. **권한 분기** — 비로그인·일반·매니저·관리자·전체
3. **데이터 패턴** — 정적 / 동적 (API 의존) / 사용자 입력 / 외부 연동
4. **모바일 우선 필요 여부** — 영업 외근 시 휴대폰 사용 페이지
5. **현재 base.html / base_sales.html / base_logi.html 중 어떤 base 사용 중인지**
