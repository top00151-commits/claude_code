## 3. 핵심 컴포넌트

1. **메가 검색 입력** (`.mega-search`) — amber border + amber-glow 4px box-shadow
2. **빅터에게 묻기 버튼** (검색 외 AI 답변 모드)

6. **우측 사이드** (최근 검색·자주 묻는 질문·팁)

## 4. Jinja2 inject map

| 시안 | inject |
|---|---|
| 검색 입력 value | `{{ q }}` |
| 결과 수 | `{{ results|length }}` |
| 응답 시간 | `{{ elapsed_ms }}` |
| 빅터 답변 헤드라인 | `{{ ai_answer.headline }}` |
| 통계 4 카드 | `{{ ai_answer.stats }}` |
| 카테고리별 결과 | `{% for r in results.by_category %}` |
| 매치 강조 | `{{ r.title|highlight(q) }}` (Jinja filter) |
| 최근 검색 | `{% for q in recent_searches[:5] %}` |
