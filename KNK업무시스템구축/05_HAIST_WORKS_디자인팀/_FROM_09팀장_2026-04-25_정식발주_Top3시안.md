- **1차 MVP 범위**: 4역할(OWNER/MANAGER/MEMBER/VIEWER) + 자원별 read/write/approve 3액션
- **UI 시사점**: R3 §4 6건
  - "보내기/회수" 두 탭 구조
  - 위임 모달 4필드(누구에게/무엇을/언제까지/사유)
  - 권한 충돌 시 토스트 + "팀장에게 요청" 버튼
  - 이력 화면 시간역순 타임라인
  - **재위임 가능 토글 OFF가 기본값** (안전한 기본값 원칙)
  - 회수 버튼 빨간색 + 2단계 확인 + Cascade 영향 미리 표시
- **페르소나**: P1 CEO(김정락), P2 팀장, P3 팀원
- **마감**: 2026-04-26 23:00
