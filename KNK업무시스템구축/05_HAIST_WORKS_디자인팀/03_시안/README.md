> **목적**: `02_개선제안_01.md` 의 제안 A (홈 3탭 분할) 을 **실제 시각**으로 확인할 수 있게 standalone HTML 프로토타입 제공.

## 파일

| 파일 | 설명 |
|---|---|
| `00_design_tokens.css` | 원칙서 §2–4 기반 CSS 변수 토큰 (color / typography / spacing) |
| `01_home_제안A_3탭.html` | 홈 제안 A — "내 업무 / 우리 팀 / 전사" 3탭 분할. role=member 기본 "내 업무" 탭 진입 예시 |
| `02_status_segment.html` | 상태 cycle 버튼 → 세그먼트 4버튼 변경 전·후 비교 |

## 여는 법

- Windows 파일 탐색기에서 HTML 파일 **더블클릭** → 기본 브라우저로 열림.
- 서버 불필요. 외부 네트워크 접근 없음 (폰트는 시스템 폴백, 아이콘은 인라인 SVG).

## 사용 범위

- ✅ 대표·09 팀장·01 세션이 **시각 확인용**으로 열어 보기
- ❌ 실제 시스템 반영 금지 — 01 세션이 수용 범위(`_TO_01_디자인제안_01.md` 회신 양식) 확정 후 코드화
- ❌ 외부 배포·공유 금지 (데이터는 목업이나 구조는 내부 시스템을 그대로 드러냄)

## 제약 준수

- 외부 CDN·폰트 API·라이브러리 로딩 없음 (대표 승인 필요 항목 0건).
- 인라인 SVG + system font 폴백만 사용.
