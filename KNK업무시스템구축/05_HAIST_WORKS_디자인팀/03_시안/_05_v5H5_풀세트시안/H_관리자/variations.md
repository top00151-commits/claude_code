## 변형 키

### A. /admin (관리자 홈)
- 5 탭 제거
- 그리드 6 카드 (회사정보·환경·권한·헬스·리마인더·외부자산)
- 각 카드: 아이콘 + 제목 + 통계 (위임 N건·미해결 M건 등) + → 진입

### B. /admin/company-info
- 폼 마스터 C 카테고리 활용
- 단계 없음, 단순 폼 (상호·사업자번호·대표·주소·로고 업로드)
- **인쇄 마스터에 영향** — 회사정보 미입력 시 빨간 배너 (G 카테고리)

### C. /admin/settings
- 그룹별 섹션 (일반·알림·언어·백업·통합)
- 각 그룹은 토글 + 입력 조합

### D. /admin/hiworks-settings
- OAuth 흐름: 연결 / 토큰 갱신 / 권한 / 해제
- 상태 칩 (연결됨·만료·해제)

### E. 위임 발송
- 폼: 사용자 선택 + 권한 선택 + 기간 + 사유
- 미리보기 (감사 로그 형태)

### F. 위임 회수
- 마스터 매트릭스 + 만료 임박 강조 (D-N 카운트다운)

### G. 감사 로그
- D 카테고리 리스트 변형
- 시간순 + 액션 칩 (위임·회수·변경·로그인)
- 필터: 사용자·기간·액션 종류

### H. 그룹 CRUD
- 트리 (그룹 → 하위 그룹 → 멤버)
- 드래그 앤 드롭

### I. 리포트 3종
- I 보고서 카테고리 활용 (다음 마스터)
- 만료 예정 / 그룹별 / 사용자별

### J. /admin/health
- 시스템 모니터링 — 서버·DB·캐시 상태 카드
- 실시간 그래프 (CPU·RAM·요청 수)

### K. 리마인더
- 시간표 — 스케줄 카드 (cron 표현)

### L. /team (팀 대시)
- 팀별 카드 + 멤버 + 통계
- 팀장이면 본인 팀만 / 관리자는 전체

### M. /team/{id}/permissions
- 마스터 매트릭스 + 팀 단위 (팀장 자기 팀만)
- 일괄 적용 (예: "이 팀 전체 매출 조회 ON")
