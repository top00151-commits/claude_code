## 마스터 핵심 컴포넌트 (모든 변형 재사용)

1. **상태 요약 5 카드** (`.summary-row`) — 클릭 시 자동 필터 (active 강조)
2. **도구바** (`.toolbar`) — 활성 필터 칩 + 필터 추가 + 검색 + 컬럼 설정 + 새로고침
3. **일괄 액션 바** (`.bulk-bar`) — 검정+앰버 글로우 (체크박스 선택 시)
4. **테이블** — 정렬 가능 헤더 + 진척 막대 + 상태 칩 + 메타정보 small
5. **페이지네이션** — 총 건수 + 페이지 버튼 + 페이지 크기 선택

## 변형 키 — 페이지별 차이 spec

### A. /feed (팀 피드)
- 상태 요약 제거
- 테이블 → **타임라인 카드** (시간순, 작성자 아바타 + 본문 + 액션)

### B. /now (실시간 진행)
- 테이블 + 진척 막대 (마스터와 유사)
- 1분 자동 갱신 (애니메이션)
- 우상단 "Live" 펄스 닷

### C. /history (내 이력)
- 상태 요약 제거 → 그룹 헤더 (오늘·어제·이번주·이전)

### D. /calendar (캘린더)
- 테이블 → 월간 그리드 (7×6)
- 일정 칩 (event-pill)

### E. /notifications (알림함)
- 그룹: 안 읽음 / 오늘 / 이번주
- "모두 읽음" 일괄 액션

### F. /board (게시판)
- 마스터 + 상단 카테고리 탭 (회사·팀)
- 이미지 thumb 컬럼

### G. /tickets (티켓)
- 마스터 그대로 + 우선순위 컬럼 추가
- 상태 칩: 신규·진행·해결·종결

### H. /customers (고객사)
- 테이블 vs 카드 토글
- 카드 뷰: 로고·이름·등급·매출·담당
- 등급 칩: VIP·A·B·C (마스터 pill 재사용)

### M-S. 자재 리스트 (8종)
- 마스터 유사 + 컬럼만 변경
- 부품: SKU 코드 + 카테고리 + 재고
- 재고 잔량: **재고 막대** (low/mid/ok 색)
- 실사: 차이 컬럼 + 진척

### T-V. 이슈·변경·QC
- 이슈: 우선순위 칩 (긴급·높음·보통·낮음)
- 변경: 영향 범위 (전사·부서·팀)
- QC: 결과 칩 (합격·부적합·재검)

## 📱 모바일 우선 spec (N-5 보강 — 01 응답서 §4-4 대응)

D 카테고리 중 **모바일 우선 페이지 4종**:
- `/notifications` (M-00-09) — 알림함 (24 페이지 중 P0)
- `/tickets` (M-00-12) — 티켓 목록 (외근 중 긴급 요청)
- (변형 키 D 캘린더) — 일정 모바일 뷰
- 향후 `/customers` 모바일 (영업 외근 — 변형 키 H)

### 모바일 변환 spec (≤ 768px)
1. **상태 요약 5 카드 → 가로 스크롤** (각 60% 너비)
2. **도구바 stacking** — 필터 칩은 가로 스크롤, 검색은 풀너비, 정렬은 드롭다운
3. **테이블 → 카드 리스트** (각 행이 카드로 변환, 핵심 3 필드 + 펼치기)
4. **일괄 액션 바 → 하단 고정 시트** (체크 시 등장, 슬라이드 업)
5. **페이지네이션 → 무한 스크롤 또는 "더 보기" 버튼**
6. **상태 칩 + 액션 → 우측 스와이프** (왼쪽 스와이프 = 빠른 액션, iOS 메일 패턴)
7. **검색 → 상단 sticky 검색 바** (스크롤 시 작아짐)
8. **Empty State** → 모바일 시 일러스트 50% 축소 + 텍스트 가운데 정렬
