## 마스터 핵심 컴포넌트

1. **상세 헤더**: ID + 상태 칩 + 제목 + 메타 4종 + 액션 버튼
2. **5단계 워크플로우** (`.workflow`) — done/active 시각화
3. **2-컬럼 콘텐츠** — 메인 (섹션 카드 4-5개) + 우측 사이드 (빅터 + 이력 + 관련)
4. **섹션 카드**: 기본 정보 (def-list 2열) / 라인 표 / 첨부 / 메모
5. **활동 이력**: 점 + 텍스트 (시간순)
6. **관련 문서**: 아이콘 + 제목 + 메타 (호버 amber-glow)
7. **빅터 분석 카드**: 검정 그라데이션 (우측 최상단)

## 변형 키

### A. 게시글 상세 (`/board/post/{id}`)
- 워크플로우 제거 (게시글은 단계 없음)
- 메인: 제목 + 작성자 + 본문 (rich text) + 첨부 + 댓글 트리
- 댓글: 들여쓰기 트리 (대댓글)
- 우측: 작성자 정보 + 관련 게시글

### B. 변경공지 상세
- 워크플로우 4단계: 작성 → 승인 → 공지 → 종결
- 영향 범위 시각화: 부서·팀·인원 (체크 매트릭스)

### C. 이슈 상세
- **SLA 타이머** 헤더 강조 (마감까지 남은 시간 카운트다운)
- 우선순위 칩 (긴급·높음·보통·낮음)
- 워크플로우: 신규 → 분석 → 처리 → 검증 → 종결

### D. 티켓 상세
- 분류 칩: admin (관리자 요청) / business (업무 협조)
- 처리 흐름: 발행 → 접수 → 처리 → 닫기
- 빅터 도움말: 유사 티켓 자동 추천

### E. 고객 상세 (`/customer/{cid}`)
- 워크플로우 제거
- 헤더에 등급 칩 (VIP·A·B·C) + 거래 시작일
- 메인 카드: 회사 정보 / 담당자 / 거래 이력 / 매출 그래프 (12개월)
- 우측: 진행 견적 / 미수금 / 빅터 분석 (이 고객 패턴)

### F. 부품 상세 (`/parts/{pid}`)
- 헤더에 코드 + 카테고리 + 재고 막대
- 메인: 사양 / 단가 이력 차트 (12개월) / 거래처 비교 표 / 발주 이력
- 우측: 재고 잔량 / 안전재고 / 회전율 / 빅터 단가 협상 추천

### G. 프로젝트 상세 5종 (gantt·burndown 등)
- 마스터 그대로 + 메인 영역 차트로 교체
- gantt: 큰 SVG 간트 차트
- burndown: 라인 차트
- milestones: 타임라인

### H. 재고 FIFO 상세
- 메인: 입고 레이어 표 (시간순 + 잔량 + 단가)
- 시각화: 적재 레이어 그래픽

### I. 권한 그룹 상세
- 매트릭스 (사용자 × 권한 토글)
- 일괄 ON/OFF

### J. 단가 시뮬·이력
- 좌: 시뮬레이터 (수량·환율·할인 입력)
- 우: 결과 (총가·마진)
