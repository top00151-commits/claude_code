## 변형 키 — 페이지별 차이 spec

### A. /home (마스터 자체)
- **3 탭**: 내 업무 / 우리 팀 / 전사 — `<div class="scope-tabs">` 그대로
- **벤토 5 카드**: 내가 진행 / 견적 / 티켓 / 미작성 일일 / 알림
- **빠른 액션 4개**: 일일업무 / 견적 / 티켓 / 출고
- **페이지 제목**: `반갑습니다, [사용자명]님` (`em` 그라데이션)

### B. /sales (매출 hub)
- **base 변경**: `extends "base.html"` → `extends "base_sales.html"`
- **3 탭 제거**: 매출은 탭 없이 단일 화면
- **벤토 변경**:
  ```
  히어로: 이번 달 매출 (KNK 레드 그라데이션 — base_sales 정체성)
  4 카드: 신규 견적 / 미수금 / VIP 거래처 / 출하 예정
  ```
- **빠른 액션 4개**:
  ```
  견적 작성 / 고객 등록 / 출하 처리 / FTA 발행
  ```
- **알림 영역**: 매출 관련 (마감 임박 견적·미수금 노화·고객 클레임 등)
- **페이지 제목**: `오늘의 매출 흐름`

### C. /dashboard·/ceo (대표 대시)
- **권한 가드**: `require(req, ['ceo','admin','executive'])`
- **3 탭 제거**: 대표는 전사 단일 뷰
- **벤토 변경**:
  ```
  히어로: 전사 매출 (큰 숫자 + 전월/전년 비교)
  4 카드: 부서별 매출 1위·이슈 건수·비용 누계·임직원 활동
  매트릭스 미니: 부서별 매출 Top 5 + 스파크라인 (B 매트릭스 마스터 표 부분 재사용)
  ```
- **빅터 dock 표시**: 전사 인사이트 (자동 분석)

### D. /logistics (자재 hub)
- **base 변경**: `extends "base_logi.html"`
- **3 탭 제거**
- **경고 배너 추가** (페이지 최상단): 재고 임계 미만 자재 N종 — KNK 레드 그라데이션
- **벤토 변경**:
  ```
  히어로: 총 자재 SKU + 임계 미만 카운트
  4 카테고리 카드: 모터·구동 / 베어링 / 금속·압출 / 전장 (각 SKU 수 + 임계 미만)
  ```
- **빠른 액션 4개**:
  ```
  발주서 작성 / 입고 처리 / 출고 등록 / 실사 조정
  ```

### E. /cockpit (코크핏)
- **권한 가드**: 임원·대표
- **레이아웃 변경**: 벤토 그리드 6×4 (24 카드) — 다중 KPI 한 화면
- **차트 추가**: 미니 라인·바 차트 5종 (매출·비용·이슈·인력·고객 추이)
- **새로고침 주기**: 1분 (다른 페이지 5분)

## 📱 모바일 우선 spec (N-5 보강 — 01 응답서 §4-4 대응)

A 카테고리 중 **모바일 우선 페이지 2종**: `/home` (M-00-01) · 빅터 dock (J 카테고리)

### 모바일 변환 spec (≤ 768px)
1. **사이드바 collapse → drawer** (햄버거 아이콘 좌상단)
2. **토픽바 6 아이콘 → 3 아이콘** (빅터·알림·사용자 / ❓·언어·나가기는 사용자 클릭 시 펼침)
3. **3 탭 (내 업무·우리 팀·전사) → 가로 스크롤 sticky**
4. **벤토 그리드 4칸 → 2칸** (히어로 카드는 100% 너비)
5. **빠른 액션 4 카드 → 가로 스크롤 카드** (각 70% 너비)
6. **알림 피드 2-컬럼 → 1-컬럼**
7. **빅터 dock → 하단 고정 floating 버튼** (탭 시 풀스크린 채팅)
8. **터치 영역 최소 44px × 44px** (Apple HIG 정합)
9. **폰트 +2px** 가독성 (12 → 14, 14 → 16)
10. **하단 고정 액션바** — 페이지별 핵심 액션 1개 (예: "+ 일일업무 작성")

### 표준 토큰 미디어 쿼리 활용
```css
@media (max-width: 768px) {
  --content-pad: 16px;
  --sidebar-w: 0px;  /* drawer로 전환 */
  --dock-w: 0px;     /* floating으로 전환 */
}
```

→ 별도 시안은 P3 (Phase 4) 권고. 현 spec 만으로 01팀 반영 가능.
