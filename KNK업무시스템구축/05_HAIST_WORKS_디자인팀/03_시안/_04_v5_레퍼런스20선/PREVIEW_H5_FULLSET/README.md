## 🔍 페이지별 빅터 메모

### ① 로그인
- **차별 포인트**: 좌측 그라데이션 패널의 **3 통계 미리보기** (견적 23 / 발주 8 / 재고 경고 3) — 로그인 전부터 "오늘 무슨 일이 있는지" 알림. 매거진 v4 의 "빈 공간" 회피.
- **로고 처리**: 흰 카드 위에 얹어 원본색 보존 (이전 v3 로고 정책 정합)
- **CTA**: 그라데이션 버튼 + 글로우 그림자 → 매거진 v4 의 단조로운 검정 버튼 대체

### ② 사용가이드
- **3단계 빠른시작 카드**: 거대한 그라데이션 숫자 (오른쪽 하단 0.4 투명도) — 매거진 v4 의 차가운 흑백 숫자 대체
- **풀콰트**: 매거진 톤 유지하되 앰버 강조 좌측 보더 → "차갑지 않은 매거진"
- **빅터 노트 (검정 배경 + 앰버 글로우)**: 5종 중 가장 강한 시각 강조 — "전문가 한 마디" 무게감

### ③ 빅터 AI 풀페이지
- **4-컬럼 레이아웃**: 사이드바 / 대화 목록 / 대화 / 컨텍스트 — 슬랙·노션 AI 표준 따름
- **메시지 컨텍스트 카드**: AI 응답 안에 데이터 카드(견적 4건 표) 임베드 → 단순 텍스트 대화 ≠ 빅터
- **빠른 작업 칩**: AI 응답 후 바로 다음 액션 클릭 가능 (단가 요청 / PDF / 분석 / 알림)
- **컨텍스트 패널 (우측)**: 대화 중 참조 문서·고객사 정보·자동 추천을 항상 표시

### ④ 매출 매트릭스
- **5종 탭** (대시·일별·월별·연간·고객별): 정책 매트릭스 5종 정의 그대로 시각화. 활성 탭은 그라데이션 강조
- **4 KPI**: 첫 KPI(이번 달 총매출)는 그라데이션 카드, 나머지는 흰 카드 — **위계 명확**
- **12 고객 테이블 + SVG 스파크라인**: 각 고객 12주 추이 즉시 가시화. 매거진 v4 의 단순 표 진화형
- **우측 인사이트 패널**: 빅터 자동 분석 (highlight 카드는 검정 배경) — 데이터 + 인사이트 분리

### ⑤ 자재 매트릭스
- **경고 배너 (KNK 레드 그라데이션)**: 매출 페이지와 다른 강한 첫인상 — "자재팀은 경고 우선"
- **4 카테고리 카드 + alert-dot 펄스 애니메이션**: 임계 미만 즉시 깜빡임
- **재고 막대 (red/orange/green)**: 매출과 차별 — 자재는 "재고 비율" 직관 시각화
- **5단계 발주 진행 (발주→확정→출고→배송→입고)**: 우측 패널, 매출 페이지의 인사이트와 차별
