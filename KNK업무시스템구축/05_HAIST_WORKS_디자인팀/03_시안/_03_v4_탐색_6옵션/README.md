## 🆕 라운드 4 — CX2 × CX3 혼합 3개 (대표 직접 지시)

매거진(CX2) 구조에 갤러리(CX3)의 정제된 텍스트 처리를 비율별로 혼합:

| 옵션 | CX2:CX3 비율 | 톤 레퍼런스 | 차별점 |
|---|---|---|---|
| **CX23a 📖 Refined Editorial** | **50 : 50** | Wallpaper · Cereal · Kinfolk | 균형 — 마스트헤드 + 1px 라인 + COVER+점 결합 |
| **CX23b 🤍 Quiet Press** | **30 : 70** | Kinfolk · Apartamento · Cereal | 갤러리 우세 — 라인 거의 없음, 활자만 우아 |
| **CX23c 📰 Curated Magazine** | **70 : 30** | Monocle · Wallpaper · GQ | 매거진 우세 — 강한 마스트헤드 + 가벼운 본문 |

**핵심 차이**:
- **CX23a**: 마스트헤드 1px + 시리프 weight 300 + COVER 라벨(작음) + 우측 빨간 점 → **균형감**
- **CX23b**: 마스트헤드 거의 없음 + 시리프 weight 200 + italic "— Cover —" + 작은 빨간 점 → **부띠끄 우아함**
- **CX23c**: 강한 3px 상단 + 시리프 weight 300 + COVER 큰 라벨(빨강) → **신문 데스크 권위감**
