상태 버튼 — Before / After

홈 화면 나의 업무 카드의 상태 변경 UI. 04 OPS-005 와 유사한 "실수 시 복귀 피로" 패턴. (H-4 제안)

✗ Before — cycle 버튼

클릭할 때마다 진행중→완료→지연→대기→진행중 순환. 의도한 상태로 가려면 최대 3회 클릭.

4월 4주차 관리팀 주간보고 초안
비품 재고 엑셀 업데이트
문제: P1 엄주영(신입)이 실수로 "진행중 → 완료" 눌렀다가 되돌리려면 "지연 → 대기 → 진행중" 3번 더 눌러야 함. 카드 여러 개 관리 시 피로 누적.

✓ After — 세그먼트 4버튼

1-클릭으로 원하는 상태 직접 지정. 실수 복귀도 1-클릭.

4월 4주차 관리팀 주간보고 초안
비품 재고 엑셀 업데이트
효과: 클릭 수 최대 3회 → 1회. 접근성(ARIA role=group) 개선. 시각적 상태 구분 명확 (색채 대비).
