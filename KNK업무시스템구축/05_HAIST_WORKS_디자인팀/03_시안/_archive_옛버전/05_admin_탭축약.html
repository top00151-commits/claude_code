📥 관리코드발행목록 업로드

업무 고도화작업 (START-END)에서 받은 관리코드발행목록.xls 파일을 업로드하면, M(자동화)·T(검사기) 두 시트가 자동 파싱되어 프로젝트·고객사가 일괄 등록/갱신됩니다.

📤 데이터 내보내기

팀·사용자·프로젝트·고객사 전체를 Excel로 다운로드.

🔐 초기 비밀번호 일괄 재생성

모든 활성 사용자의 비밀번호를 knk + 4자리 랜덤숫자로 재생성하고, 팀별로 정리된 배포용 Excel을 다운로드합니다.

⚠ 주의: 재생성 즉시 기존 비밀번호는 무효화됩니다. 실사용 킥오프 시 1회만 실행하세요.

👥 권한 정책 점검

현재 물류 권한 부여: CEO·admin·executive 자동 / leader·member 수동 토글.
미부여 member 있으면 자재 모듈 접근 불가 (04 OPS-001).

⚠ 정말 전체 비밀번호를 재생성하시겠습니까?

이 작업은 즉시 적용되며 되돌릴 수 없습니다. 모든 직원이 새 비밀번호를 받기 전까지 로그인할 수 없습니다.

📐 시안 #05 주석

  • 탭 3분할: 대시보드·데이터·보안 (F-17)
  • 4 테이블 → 요약 카드 + "목록 관리 →" 링크
  • 그라디언트 카드 제거, 단색 통일 (F-19)
  • 비번 재생성 2단계 확인: 체크박스 + 대표이사명 입력
  • 보안 탭 별도 분리: 실수 방지
  • 외부 자산 0건
