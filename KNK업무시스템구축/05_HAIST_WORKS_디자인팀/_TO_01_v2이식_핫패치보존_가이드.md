## 2. 이식 절차 (권장)

```
[STEP 1] v2 시안 c2 base.html / style.css 백업
[STEP 2] LIVE base.html / style.css 의 H1~H4 라인 위치 마킹
        → 주석으로 "// HOTPATCH-c2-H#" 태깅 권장
[STEP 3] v2 시안 통째 이식 (덮어쓰기)
[STEP 4] H1~H4 라인 재이식 (재적용)
[STEP 5] grep15 양쪽(시안·LIVE) 동시 실행 — G01~G15 전부 PASS
[STEP 6] 04 힐링 QA 2차 의뢰
```

## 3. 회피 사항 (Anti-pattern)

- v2 시안만 적용하고 H1~H4 재이식 누락 → 빅터도크 / WORKSPACES alias 다시 깨짐
- 핫패치 라인을 v2 시안 본문에 **직접 끼워 넣기** → 시안-LIVE 정합 다시 깨짐
- 이식 후 시안 grep만 수행 (어제 사고 재발)
