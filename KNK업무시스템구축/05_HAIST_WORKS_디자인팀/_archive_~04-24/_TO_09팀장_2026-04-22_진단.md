## 1. 현 사이드바 메뉴 실측 (2 depth 기본)

### 현황
- **섹션 12개** (본문 라인 66–212): `내 업무 → 전사 → 진행률 → 요청 → 이슈 → 변경 → 게시판 → 외부 → 매출 → 자재 → 팀/프로젝트 → 관리자`
- **Depth 구조**: 섹션 제목(depth 0) → 링크 항목(depth 1) → 서브트리(팀/프로젝트 동적 depth 2)
- **외부 링크** 2개 하드코딩(`hiworks_approval_url`, `hiworks_mail_url`) + 2개 추가 필요(자금·메신저)
- **권한 조건**: leader/executive/ceo/admin 별로 cockpit/bottleneck/dashboard/admin 숨김 구현

### 분석
- 12개 섹션은 "과다" (원칙 §5-1 권장 6개)
- 섹션 헤더에 배지(`매트릭스 | 카톡 대체 | 학습 | 실시간 | 하이웍스` 등)로 용도 구분 → 신입 멘탈모델 복잡
- 매출/자재 섹션은 `target="_blank"` 외부 진입점 → "내부" vs "외부" 시각 혼동 가능

## 4. 외부 유지 4건 링크 노출 패턴 (2안)

> **현황 문제**: 자금·메신저는 아직 하드코딩 미구현. 메일·전자결재는 `{{ hiworks_approval_url }}`, `{{ hiworks_mail_url }}` 변수로 노출 중.  
> **디자인 원칙 위반**: 외부 링크와 내부 모듈 시각 구분 미흡 → 신입 혼동 위험 (원칙 §5-2 "클릭 예측").

### **안 1: 섹션 헤더 시각적 구분**

```
🔗 외부 연결 — [아이콘 배지 "하이웍스 외부"]
  ├─ 💰 자금 시스템 (외부링크, target="_blank", ↗ suffix)
  ├─ 📧 메일 (외부링크, target="_blank", ↗ suffix)
  ├─ 💬 메신저 (외부링크, target="_blank", ↗ suffix)
  └─ ✔️ 전자결재 (외부링크, target="_blank", ↗ suffix)
```

**특징:**
- 섹션 헤더에 "🔗 외부 연결" 명시 + "외부" 배지 (배경 #EEF2FF, 색 #4338CA)
- 모든 4개 항목에 `target="_blank"` + 우측 화살표 suffix ("↗")
- 마우스 호버 시 배경 연한 회색 (`#F3F4F6`)으로 변경 → "클릭하면 나감" 신호
- 현재 코드(base.html 177–188): 메일·전자결재 이미 구현 → 자금·메신저만 추가

**구현 비용**: base.html 내 `hiworks_approval_url`, `hiworks_mail_url` 변수 2개 추가 필요. (코드 수정 최소화)

---

### **안 2: 드롭다운 아코디언 (숨김 기본)**

```
🔗 외부 연결 [▸ 더보기] ← 기본 닫힘
  └─ [▾ 펼쳐짐] 클릭 시:
     ├─ 💰 자금 시스템
     ├─ 📧 메일
     ├─ 💬 메신저
     └─ ✔️ 전자결재
```

**특징:**
- 기본 닫힘 상태 → 사이드바 높이 줄임 (스크롤 부하 감소)
- 클릭 시 4개 링크 펼침 (아코디언)
- "내부" vs "외부" 명확한 시각 분리
- 신입이 "외부 시스템은 따로 있구나" 인식 용이

**구현 비용**: CSS `display:none` + JS 토글 필요 (현 base.html 모달 방식과 유사). 코드 수정 포함.

### **권장**: **안 1** (섹션 헤더 구분)
- 현재 코드와 호환성 높음 (변수만 추가)
- 외부 링크를 "숨지 말되 명확히 표시" (접근성·신뢰도 ↑)

## 5. 페르소나 근거 3명

### P1 (신입 개발자 · 한국 · 26세)
- **현황**: `/home` 방문율 70% (매일 아침 대시보드 확인)
- **어려움**: 
  - 사이드바 12개 섹션을 훑으며 "내가 해야 할 일이 어디 있지?" 10초 이상 소요
  - `/feed` 필터 칩 9개에 "이 화면이 뭔지 모르겠는데" 반응 (현황파악 H-1)
  - 외부 링크(메일·전자결재)를 클릭했을 때 "새 탭이 열렸네?" → 하이웍스라는 것을 30초 후 인식
- **안 1의 효과**: 네비 5개 섹션으로 축약 → 한눈에 카테고리 파악 · "🔗 외부 연결" 명시 → 외부 인식 즉시

### P6 (팀장 · 한국 · 38세)
- **현황**: `/dashboard` + `/cockpit` + `/team` 순회 (경영 현황 추적)
- **어려움**:
  - 변경 알림·병목·코크핏이 "전사" 섹션에 분산 → "변경 사항 어디서 보지?" 혼선

  - 외부 링크 화살표 없으면 "이게 여기 시스템인 줄 알았어" (UX 혼동)
- **안 1의 효과**: "🔗 외부 연결" 명시 + 4개 모두 `target="_blank"` + 화살표 → 시각 신호 명확 · 베트남어 번역도 단순화 ("외부")
