### 2.1 실제 적용 확인 (database.py 738~752)

```
738 -- 재고 잔고 VIEW (시안 §데이터 모델 — stock_balances MATERIALIZED VIEW · SQLite는 일반 VIEW)
739 -- on_hand = sum(IN) - sum(OUT) - sum(ADJUST 부호 합) · idempotent
740 -- 사이클 52 (2026-04-27) 결함 수정: parts.name → parts.part_name (실제 컬럼명 정합).
741 -- DROP VIEW IF EXISTS + CREATE VIEW 패턴으로 기존 DB도 자동 재생성 (마이그레이션 안전).
742 DROP VIEW IF EXISTS stock_balances;
743 CREATE VIEW stock_balances AS
744 SELECT  p.id              AS part_id,
745         p.part_no         AS part_no,
746         p.part_name       AS part_name,
747         COALESCE(SUM(sm.quantity), 0) AS on_hand,
748         p.unit            AS unit,
749         MAX(sm.occurred_at) AS last_movement_at
750 FROM    parts p
751 LEFT JOIN stock_movements sm ON sm.part_id = p.id
752 GROUP BY p.id, p.part_no, p.part_name, p.unit;
```
