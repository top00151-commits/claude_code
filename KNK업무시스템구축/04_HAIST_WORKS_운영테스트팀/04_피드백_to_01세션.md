## 🟠 S2

### [01-ACT-B] `/api/task` 서버측 입력검증 *(OPS-002)*

**문제**: 빈 제목, 음수 시간, 40h/일 모두 200 OK 저장. 대시보드 오염.

**수정 제안** (`app/main.py:441` 전후):
```python
title = (d.get("title") or "").strip()
if not title:
    return JSONResponse({"error":"제목은 필수입니다."}, 400)
hours = float(d.get("hours") or 0)
if hours < 0 or hours > 24:
    return JSONResponse({"error":"업무 시간은 0~24시간 범위여야 합니다."}, 400)
status = d.get("status") or "진행중"
if status not in STATUSES:
    return JSONResponse({"error":f"상태 값 오류: {status}"}, 400)
category = d.get("category") or "기타"
if category not in CATEGORIES:
    return JSONResponse({"error":f"카테고리 값 오류: {category}"}, 400)
```

**또한**: `PUT /api/task/{tid}` 에도 동일 검증.
