### 2-1 페르소나 프로필
- 역할: 영역 10 수출 담당 (CO/PL/Invoice 작성)
- 소속: KNK 베트남 법인 (team_id=12)
- 연차: 현지 경력 2~3년
- 기본 언어: 베트남어
- 핵심 페인포인트:
  - UI 한국어 의존 → VI/EN 번역 필수
  - 서류가 국내법·베트남법 이중 요구
  - 시차 제약 (베트남 현지시간, 한국팀과 비동기)

### 2-2 영역 10 수출입 6단계 흐름
1. 매출 대기 (P4) → CO 작성 신호
2. P11 CO 작성 (공급자/구매자/Item/Qty/Price) + Order No. 자동
3. CO 검사·서명 → PDF 출력 → 이메일 발송. DB "Sent"
4. 제조 진행 추적 (Daily 카드 연동)
5. P11 PL 작성 (포장 상자수·무게·용적)
6. P11 Invoice 작성 (세관용 VI 병행)
7. 완료: DB "Exported" → CEO Cockpit 월별 수출액

각 단계 시간: CO 15분 + PL 15분 + Invoice 10분 = ~40분

### 2-3 다국어 UI 기대값
- VI: "주문 확인서" → "Xác nhận đơn hàng"
- EN: "Confirmation of Order"
