## 2. 수정 요청

### 대상 파일

`app/main.py` — `/api/translate` 라우트

### 요청 내용

`/api/translate` 엔드포인트 호출 시 아래 항목을 **DB `translation_log` 테이블 또는 앱 로그**에 기록

| 기록 항목 | 설명 |
|-----------|------|
| `user_id` | 요청한 사용자 ID |
| `source_lang` | 원본 언어 코드 (예: `ko`, `en`) |
| `target_lang` | 번역 대상 언어 코드 |
| `timestamp` | 호출 일시 (UTC 또는 KST) |
| HTTP 응답코드 | 번역 API 응답 상태 (200, 4xx, 5xx 등) |

### 구현 방식 (01 메인 재량)

아래 중 구현 편의에 맞는 방식 선택:
- **DB 방식**: `translation_log` 테이블 신규 생성 후 INSERT
- **파일 로그 방식**: 기존 앱 로거(`logging` 모듈 등)에 INFO 레벨로 기록

어느 방식이든 위 5개 항목이 모두 포함되면 04 팀에서 집계 가능.
