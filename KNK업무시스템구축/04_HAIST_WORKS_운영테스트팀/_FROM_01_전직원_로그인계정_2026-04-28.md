## 2. 통합 비밀번호 (84명 공통)

```
PASSWORD: knk2026!
```

- 해시 방식: `sha256("knk-haist-" + pw)` (서버 `hash_pw()` 함수)
- 서버 측 저장: `data/knk.db` `users.password` 컬럼
- 적용 시각: 2026-04-28 (백업 미생성 — 테스트용)

## 3. 사용 방법

### 3-1. 브라우저 (UI 테스트)
1. http://localhost:8081/login 접속
2. 아이디 입력: 한글 직접 입력 (예: `안지연`) 또는 ASCII (예: `kjr`)
3. 비밀번호: `knk2026!`
4. 로그인 → 역할별 자동 redirect

### 3-2. requests / Python (HTTP 시뮬레이션)
```python
import requests
s = requests.Session()
r = s.post('http://127.0.0.1:8081/login',
           data={'login_id':'안지연', 'password':'knk2026!'})
# r.status_code == 303 · r.headers['location']로 redirect 확인
```

### 3-3. curl (한글 ID는 URL-encoded form-data 권장)
```bash
curl -i -c cookies.txt -X POST http://localhost:8081/login \
  --data-urlencode "login_id=안지연" --data-urlencode "password=knk2026!"
```
