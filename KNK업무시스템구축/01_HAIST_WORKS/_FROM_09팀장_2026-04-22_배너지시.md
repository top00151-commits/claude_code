## [D01-NEW-BANNER] 권한 없음 안내 배너 추가

**대표 결재 원문**: "B안 채택 — '이 메뉴는 권한이 없습니다. 구매팀장에게 요청하세요' 안내 배너 추가"

### 현재 상황
권한 없는 직원이 막힌 페이지(예: `/logistics`)를 클릭하면 아무 설명 없이 홈으로 리다이렉트됨.
→ URL 파라미터 `?no_perm=logistics` 형태로 이미 홈에 전달되고 있을 가능성 있음 (코드 확인 필요).

### 구현 지시

1. `/home` 라우트 또는 `home.html` 에서 `no_perm` 쿼리 파라미터 감지
2. 파라미터 존재 시 상단 배너 표시:
   - **문구**: "이 메뉴는 권한이 없습니다. 구매팀장에게 요청하세요."
   - **스타일**: 경고(주황) 계열, 3~5초 후 자동 소멸 또는 X 클릭 닫힘
   - **연동**: D01-NEW-PERM(구매팀장 권한 관리 UI) 완성 후 링크 추가 가능하나, 현재는 텍스트 메시지만
3. `no_perm` 파라미터가 없는 경우 기존 홈 화면 그대로 유지 (사이드이펙트 없음)
4. 커밋: `권한 없음 안내 배너 추가 (대표 결재 2026-04-22)`
