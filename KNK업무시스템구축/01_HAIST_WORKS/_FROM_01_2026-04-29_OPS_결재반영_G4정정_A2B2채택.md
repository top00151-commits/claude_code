### 구현 내용 (`app/main.py` 신규 4개 함수 + 1개 엔드포인트)

| 항목 | 위치 | 역할 |

| `_seconds_until_next_1630_weekday()` | line 88 | 다음 평일 16:30까지 남은 초 (토·일 자동 건너뜀) |

### 동작 시나리오

1. uvicorn 기동 시 `startup()` → `_start_daily_reminder_scheduler()` 자동 호출
2. 콘솔 로그: `[DAILY-REMINDER] 스케줄러 시작. 다음 발송까지 N초 (M시간).`
3. 평일 16:30 도달 → SQL 쿼리로 미작성자 추출 → `notify_user()` 반복 호출
4. 콘솔 로그: `[DAILY-REMINDER] 2026-04-29 → 미작성 5명 / 알림 5건 발송`
5. 같은 함수가 다음 평일 16:30 자동 재예약
6. 사용자가 다음 로그인 시 우상단 🔔 (notifications.html) 에 알림 표시 → 클릭 시 `/daily` 이동

### 중복 방지 (database.py:3078)

`notify_user()` 내장 — 같은 user+kind+title 1시간 내 중복 INSERT 차단. uvicorn 재시작·수동 트리거 다중 호출 시에도 알림 폭주 X.
