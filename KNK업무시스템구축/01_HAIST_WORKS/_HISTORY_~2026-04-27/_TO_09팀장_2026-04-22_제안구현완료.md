## 수정 파일 상세

### A. `app/templates/progress_matrix.html`
- 매트릭스 테이블 위에 `📊 매트릭스 / 🃏 카드뷰` 토글 버튼 추가
- 기존 테이블에 `id="progMatrixView"` 부여
- 카드뷰 `id="progCardView"` 추가: 12공정 상태 뱃지 + 전체율 + 납기 + 클릭 시 `/progress/{id}`
- CSS: `.pvtab`, `.pvtab-on`, `.prog-card` hover 효과
- CSS `@media (max-width:767px)`: 모바일에서 자동으로 카드뷰 전환, 토글 버튼 숨김
- JS: `switchProgressView()` + `localStorage` 뷰 설정 유지

### B. `app/templates/feed.html`
- 필터 바에서 `chip-rk` (hasrisk) 및 `chip-ment` (mentionme) 2개 버튼 제거
- **이유**: alert-band의 ⚠️ 리스크 / ❓ 내게 멘션 버튼과 완전 중복
- alert-band 버튼은 유지 (동일 `data-filter` 연동으로 필터링 기능 정상 작동)
- 삭제된 버튼 위치에 주석 추가 (이유 명시)

### C. `app/templates/admin.html` — **전체 재구성**
- 3탭 네비게이션 추가 (`admin-tab-nav`): 📊 대시보드 / 📥 데이터 / 🔐 보안
- **대시보드 탭**: KPI 4카드 + 팀 테이블 + 사용자 테이블 + 프로젝트 테이블 + 고객사 테이블
- **데이터 탭**: 관리코드발행목록 업로드 (기존 그대로)
- **보안 탭** (ceo/admin only): 시스템 설정·건전성 링크(그라디언트 → 단색 카드) + 비번 재생성 **2단계 확인**
  - 비번 재생성: 1클릭 → 확인 패널 → 체크박스 동의 → 실행 (OPS-008 보안 강화)
- 탭 상태 `localStorage` 유지 (`adminTab_v2`)
- "물류" 컬럼 → "자재 권한" (F-18 이슈 해소)

### D. `app/main.py`
- `role_home(user: dict) -> str` 헬퍼 추가 (ESC-02)
  - ceo/admin/executive → `/dashboard`
  - leader → `/team`
  - member/그 외 → `/home`
  - 미인증(None) → `/login`
- `/dashboard` 라우트 ESC-02 수정:
  - 기존: `require()` 실패 시 무조건 `/login`
  - 수정: `get_user()` 로 로그인 상태 먼저 확인 → 미인증만 `/login`, 권한 부족 로그인 상태 → `role_home()` 호출
