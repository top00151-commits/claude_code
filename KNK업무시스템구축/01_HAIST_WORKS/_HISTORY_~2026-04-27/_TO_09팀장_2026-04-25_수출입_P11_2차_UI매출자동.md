## 4. 시퀀스 패턴 (CI-/PL- · idempotent)

```python
def _next_seq_no(c, table, col, prefix):
    ym = datetime.now().strftime("%Y%m")
    row = c.execute(f"SELECT {col} FROM {table} WHERE {col} LIKE ? "
                    f"ORDER BY {col} DESC LIMIT 1", (f"{prefix}-{ym}-%",)).fetchone()
    seq = int(str(row[col]).rsplit("-",1)[-1]) + 1 if row and row[col] else 1
    return f"{prefix}-{ym}-{seq:04d}"   # CI-202604-0001 / PL-202604-0023
```

트랜잭션 내 호출 (`with db_session()`) — INSERT 와 동시 commit, race 방어.
