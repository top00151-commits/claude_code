## 1. 신규 테이블 6개 (`app/database.py` SCHEMA)

idempotent 가드 (CREATE TABLE IF NOT EXISTS) 전건 적용. 시안 §데이터모델 인용.

| # | 테이블 | 핵심 컬럼 | 비고 |
|---|---|---|---|
| 1 | `permissions` | id / name(UNIQUE) / scope / created_at | 권한 카탈로그 |
| 2 | `permission_groups` | id / name(UNIQUE) / description / created_at | Group Inheritance ③ |
| 3 | `group_permissions` | group_id + permission_id (PK 복합) | 그룹↔권한 다대다 |
| 4 | `user_groups` | user_id + group_id (PK 복합) | 사용자↔그룹 다대다 |
| 5 | `delegation_tokens` | id / from_user / to_user / permission_id / expires_at / can_redelegate(0=OFF기본) / status(ACTIVE/EXPIRED/REVOKED) / created_at | Delegation Token ④ |
| 6 | `delegation_audit` | id / token_id / action(GRANT/DELEGATE/REVOKE/EXPIRE) / actor / timestamp / details | append-only 감사로그 |

추가 인덱스 4종: `idx_dt_to`, `idx_dt_from`, `idx_da_token`, `idx_da_actor`.
