### 커밋 2: `ee86f67` — 요청 티켓 시스템

**설문 매핑**: 1순위 ③ / 10팀 공통 (카톡·구두 요청 누락 해결)

**구현**:
- DB 2개 테이블: `tickets` / `ticket_comments`
- 자동 채번: `TKT-YYMMDD-NNN`
- 카테고리별 자동 라우팅
  * 자재요청 → 구매팀 / 긴급가공 → 가공팀 / MODIFY → 설계팀
  * 검수요청 → 품질팀 / AS → 검사기·자동화팀 (사업부별)
- 4-step 워크플로우 폼 (종류→내용→수신/일정→등록)
- 상태 워크플로우: 요청 → 접수 → 처리중 → 완료/반려
- 1클릭 상태 변경 + 자동 시스템 메시지
- 요청자↔수신자 대화창 (댓글)
- 출처(source) 추적: web/카톡/메일/전화
- 사이드바 미처리 카운트 파란 뱃지 폴링

**검증**: TKT-260420-001 (자재요청) → 구매팀 자동 라우팅 확인

### 커밋 3: `89b6d23` — 진행률 대시보드

**설문 매핑**: 1순위 ① / 8팀 공통 (모든 부서 매일 아침 1순위 정보)

**구현**:
- DB 1개 테이블: `project_phases`
- 12공정 표준: 수주→컨셉→설계→전장→SW→가공→구매→조립→검수→출하→Set-Up→KNKVN
- 자동 부서 매핑 (사업부별)
  * design→설계팀 / elec→전장설계팀 / sw→SW팀
  * machining→가공팀 / buying→구매팀
  * assembly→제조1(T)/제조2(M) / qc→품질팀
- Lazy 시드: 프로젝트 첫 조회 시 12공정 자동 생성
- 매트릭스 뷰: 관리코드(50) × 12공정 (색상 셀)
- 셀 클릭 → 상세 12공정 카드 그리드
- 1클릭 액션: [시작] [완료] [지연] (자동 진척률·일정 갱신)
- 진척률 직접 입력 폼 + 메모

**검증**: 50개 관리코드 매트릭스, 첫 프로젝트 12공정 카드 + 액션 버튼 66개
