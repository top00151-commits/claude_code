## ✅ 매일 하는 일

### ① 출근 후 5분 — 홈 확인
- `/home` → 내 할 일 / 확인 안 한 변경 / 받은 티켓 / 지연 공정
- 빅터에 "내 할 일" 물어보면 요약 즉시 표시

### ② 일일업무 카드 작성
- `/daily` → 오늘 할 일 카드 등록 → 진행 상태 업데이트
- 완료 시 "완료" → 자동으로 팀장/부서 피드에 반영

### ③ 변경 사항 확인 (빨간 뱃지 있을 때)
- 사이드바 **변경 목록** → 내 부서에 영향 주는 변경 확인
- "✓ 확인했습니다" 클릭 → 변경 공지 사이클 완료

## 🎫 자주 쓰는 작업별 가이드

### "다른 팀에 요청할 게 있어요" → 요청 티켓
- `/tickets/new` → 카테고리 선택 (자재요청 / 긴급가공 / MODIFY / AS 등)
- 자동으로 적절한 팀에 라우팅됨
- 결재 필요하면 → 하이웍스에서 결재 후 URL 첨부

### "설계/자재/프로세스가 바뀌었어요" → 변경 Inform
- `/changes/new` → 4단계 폼 (종류 / 사업부 / 내용 / 공지)
- 영향 부서 자동 판별 → 모든 영향자에게 알림
- 외부 CAD 출처 선택 가능 (AutoCAD/SolidWorks/Inventor)

### "고객사 이슈·AS가 발생했어요" → 이슈 DB
- `/issues/new` → 심각도 / 종류 / 고객사
- 원인분석 → 조치 → **재발방지 대책** (회사 자산!)

### "자재 출고해야 해요"
- `/stock/issue` → 부품 선택 → 현재고 확인 → 수량 / 프로젝트 지정
- 또는 `/parts` 에서 "📤 출고" 버튼

### "자재 입고됐어요"
- `/po` → 해당 발주서 → **📥 입고 처리** 버튼
- 라인별 입고 수량 입력 → 재고 자동 증가

### "재고 실사했어요"
- `/stock/adjust` → 차이 입력 (+/-) → 사유 필수

### "수불부 확인하고 싶어요"
- `/stock/movements` → 전체 입출고 이력 + 필터
- 부품별 이력: `/parts` → "📒 이력" 버튼

### "이번달 매출이 어떻게 되죠?"
- 🤖 빅터에 **"이번달 매출"** 또는 **"매출 현황"** 입력
- 즉시 당월/YTD/사업부별 집계 표시

### "리니어가이드 재고 얼마나 있어?"
- 🤖 빅터에 **"리니어가이드 재고"** 입력
- 즉시 검색 결과 + 안전재고 경고 표시

## 🔔 알림은 어디로?

| 종류 | 표시 위치 |
|------|-----------|
