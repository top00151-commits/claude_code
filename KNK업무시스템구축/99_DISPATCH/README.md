> **운영자**: 09 프로젝트팀장 (빅터)

## 목적
모든 중요 이벤트를 **한 곳**에서 시간순 누적. 대표가 자리 비웠다 돌아오시면 이 폴더만 보면 됨.

## 채널 구성
1. **PushNotification (실시간)** — 데스크톱 알림 + 모바일 (Remote Control 활성 시)
2. **99_DISPATCH 폴더 (영구 기록)** — 한 이벤트 = 한 .md 파일

## 파일명 규칙
`YYYY-MM-DD_HH-MM_<대분류>_<요지>.md`

대분류:
- **TIER** — Plan Y 단계 완료 알림 (Tier 1/2/3)
- **S1** — 현업 차단 발견
- **AUDIT** — 감사팀 "거짓" 또는 "부분거짓" 판정
- **EXT** — 외부 시스템 연결 필요
- **DEC** — 대표 결정 필요 사항
- **FYI** — 단순 진척 보고

## 발사 기준 (대표 응답 기다리는 것 vs 보고만)

| 분류 | PushNotification | 폴더 기록 | CEO 응답 필요 |
|---|---|---|---|
| TIER 완료 | ✅ | ✅ | ❌ (확인만) |
| S1 차단 | ✅ | ✅ | ✅ "진행해/대기해/하지마" |
| AUDIT 거짓 | ✅ | ✅ | ✅ |
| EXT 외부 연결 | ✅ | ✅ | ✅ |
| DEC 결정 | ✅ | ✅ | ✅ |
| FYI 진척 | ❌ | ✅ | ❌ |

## 대표 응답 표준 (4 단어)
- **"진행해"** = 승인
- **"추가더검토해"** = 보강 후 재제안
- **"대기해"** = 일시 보류
- **"하지마"** = 거부

## 누적 인덱스
첨부 파일들 시간순 자동 정렬됨. 가장 최근 = 맨 아래.
