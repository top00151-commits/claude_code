  - deep_translator: **removed_2026-04-27_cycle69** ✓
- 외부자산 0 정책 100% 정합

### B. 회사 정보 (KNK) ⭐ 대표 직접 입력 사안
- 사이드바 **"🏢 회사 정보 (KNK)"** 또는 **관리 → 회사 정보**
- **10 필드 직접 입력**:
  1. 회사명 (한국어): "주식회사 케이엔케이" 등
  2. 회사명 (영문): "KNK Co., Ltd."
  3. 사업자번호
  4. 주소 (한국어)
  5. 주소 (영문)
  6. 대표 전화
  7. 대표 이메일
  8. 대표자명 (한국어): "김정락"
  9. 대표자명 (영문): "Jeongrak Kim"
  10. 팩스 (선택)
- 입력 후 **저장** 클릭 → 견적서 인쇄 시 자동 반영
- **비어있으면** 견적서에 `[등록 대기]` 빨간 글씨

### C. 견적서 작성 + 다국어 인쇄 (사이클 58·63)
- **매출·영업 Hub → 견적**
- 견적 작성 → 라인 추가 → "인쇄 보기" 클릭
- **언어 라디오** (한국어 / English / Tiếng Việt) — 사내 사전 530+키 활용
- 인쇄 화면에서 회사 정보·서명란·라인 표 + 직인란 자동 표시
- 인쇄: 브라우저 `Ctrl+P`

### D. CEO 대시보드
- 통합 메인 → **6 패널** (KPI):
  1. 매출 KPI
  2. 진행률
  3. 이슈
  4. 결재 / 요청
  5. 안전재고 알림 (사이클 53 신설) — 부족 부품 Top 5
  6. 미확인 알림

### E. 자재·구매 (Logistics Hub)
