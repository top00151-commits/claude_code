# 사용법 — 10 경영현황 대시보드

> **대상**: 대표이사·임원·관리자
> **목적**: 3개 모듈(검사기 완제품·자동화 완제품·소모품) 통합 조회
> **기준일**: 2026-04-15 (V4 배포본)

---

## 1. 폴더 구성

```
10_대시보드/
└── KNK_경영현황_2026.xlsx
    ├── [매출총괄]      ← 유형별 한눈에 보기
    └── [프로젝트목록]   ← 전체 프로젝트 검색·필터
```

---

## 2. 매출총괄 시트 (요약 KPI)

| 컬럼 | 의미 |
|---|---|
| 유형 | 검사기 완제품 / 자동화 완제품 / 소모품 |
| 진행 | 활성 프로젝트 수 (수주예정·진행중·보류) |
| 완료 | 납품완료 건수 |
| 총 | 진행 + 완료 합계 |
| KRW | 원화 수주 합계 |
| USD | 달러 수주 합계 |
| INV$ | 인보이스 USD 환산 합계 |
| 진척률 | 활성 프로젝트 평균 진척률 |
| 지연 | 납기 지난 활성 프로젝트 수 (D-day < 0) |

**합계 행**: 모든 모듈을 통합한 전사 수치

---

## 3. 프로젝트목록 시트 (상세 검색)

| 자동 표시 |
|---|
| 유형, 관리코드, 수주번호, 고객사, 품명, 통화, 금액, 진척률, 납기일, D-day, 진행상태, 등록일자 |

**활용 방법**:
- Excel 필터 기능으로 고객사별·상태별 검색
- 진행상태 = 지연인 항목만 추출 → 즉시 대응 회의 자료
- 통화 = USD인 항목만 추출 → 환율 영향 모니터링

---

## 4. 갱신 방법 (대시보드 자동 재생성)

01/02/05 모듈에서 sync 실행 후, 대시보드를 갱신합니다.

```cmd
cd 10_대시보드
python scripts/gen_dashboard.py
```

자동 처리:
1. 01/02/05 각 모듈에서 최신 데이터 수집
2. 매출총괄·프로젝트목록 시트 재계산
3. 새 파일로 저장

**실행 시점**:
- 매주 월요일 오전 (주간보고 전)
- 월말 결산 시
- 주요 수주·납품 발생 시 즉시
