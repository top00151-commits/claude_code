## 🎯 V1 달성 기능 (설문 대조 관점)

### ✅ 완료

#### 1. 엑셀 PMS 4개 모듈 정규 구조
- **01 검사기 완제품** (31열 PMS + 8부서 입력 파일)
  - 23열 ID(제품구분 포함) + 8부서 담당자
  - 제품구분 드롭다운: PBA / TSP / SENSOR / 기타
- **02 자동화 완제품** (29열 PMS + 7부서 입력 파일)
  - 22열 ID + 7부서 담당자 (v2026.04 환율·장비출하·셋업·이슈사항 제거)
- **05 소모품** (소모품목록 10열 + 출하관리 40열)
- **10 대시보드** (경영현황 2026.xlsx)

#### 2. 스펙 §20 체크리스트 20항 전부 자동 보장
- R1 좌측 indent=1 + 32px 높이
- R2 좌측 슬로건 병합 + 우측 업데이트 일시
- R3 가이드 12유형 (자동·입력·선택·PO후입력 등)
- R4 헤더 영역 색상 + **체계적 메모 500×220 px**
- R5+ 데이터 9pt 맑은 고딕 + 교대 배경
- 컬럼별 정렬 (수량 중앙·금액 우측·품명 좌측 indent=1)
- 통화별 금액 포맷 (₩#,##0 / $#,##0.00)
- **글틀고정 "품명까지 고정" 자동 계산** (freeze=auto)
- 컬럼 너비 데이터 기반 자동조정 (한글 2폭, min 5/max 40)
- 빈 행 자동 삭제 (Ctrl+End이 마지막 데이터 행으로)
- D-day 역카운터 `+0"일";-0"일";"0일"` + 납품완료 공란
- 정렬 우선순위: 진행중 → 제안단계 → 보류 → 취소 (납품완료 제외)
- 시트 보호 (비밀번호 `knk2026`)
- VML 메모 박스 500×220 px 강제

#### 3. 자동화 기능
- **관리코드 자동 채번** — `{SEQ:03d}{BIZ}{YYMM}` (예: 001T2604)
- **수주번호 자동 채번** — `{BIZ}-{YYMMDD}[-N]`
- **날짜 정규화** — 2026/4/15, 260415 등 → 2026-04-15
- **금액 자동 계산** — 수량 × 단가 (통화별 포맷)
- **부서 양방향 동기화** — PMS 부서 컬럼 ↔ 부서 파일 C10 담당자
  - "제외" 선택 시 해당 부서 파일에 미등록
  - 부서 파일에서 이름 입력 시 PMS에 역반영
- **정렬 + 상단 10칸 빈 공간 제거** (v2026.04c 이후)
- **납품완료 자동 아카이브** → 4_완료이력

#### 4. 사용자 편의
- **`.bat` 18개** — 부서별 + PMS 전용 (더블클릭 한 번으로 sync+apply_standard)
- **상세 헤더 메모** — 각 컬럼 클릭 시 용도·포맷·예시 설명
- **PO유형 확장** — 신규/추가/수정/개조/기타
- **일정표 임포트** — `98_문서/import_schedule.py`로 203건 자동 분류·임포트

## ⏮️ 원복 방법

V2 진행 중 문제 발생 시 원복:

```bash
# 1. 현재 HAIST_WORKS_baby 폴더를 백업 (혹시 모를 대비)
Rename-Item "HAIST_WORKS_baby" "HAIST_WORKS_baby_v2_failed_YYYYMMDD"

# 2. V1 복사
Copy-Item "HAIST_WORKS_baby_v1" "HAIST_WORKS_baby" -Recurse
```

**CMD 버전**:
```cmd
rename HAIST_WORKS_baby HAIST_WORKS_baby_v2_failed_YYYYMMDD
xcopy HAIST_WORKS_baby_v1 HAIST_WORKS_baby /E /I /H
```
