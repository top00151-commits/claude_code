### 대안 ①. 스마트공장 수준확인 사업 ⭐ 1순위 후보

**공고 번호**: `PBLN_000000000116004` (2026 스마트 제조혁신 지원사업 통합공고 내)

#### 1-1. 공식 자격 요건 (팩트)

다음 세 조건 중 하나 충족 시 신청 가능:

| 조건 | 내용 | KNK 해당 여부 |
|---|---|---|
| 조건 A | **자체 역량으로 스마트공장을 구축한 기업** (정부 스마트공장지원사업 **미참여**) | 🟢 **가능성 매우 높음** |
| 조건 B | 자발적으로 고도화를 추진하여 스마트화 수준상승이 기대되는 기업 | 🟢 가능 |
| 조건 C | 스마트공장 수준확인서의 유효기간 만료기업 | ❌ 해당 없음 (미수령) |

#### 1-2. 지원 내용

| 항목 | 내용 |
|---|---|
| 지원금 | **수준확인 비용 전액 지원** |
| 결과물 | 수준확인서 + 진단보고서 |
| 목적 | 중소·중견 제조기업의 스마트화 수준 진단 |

#### 1-3. 스마트공장 레벨 정의 (공식)

| 레벨 | 내용 |
|---|---|
| **ICT 미적용** | 자동화 없이 수기로 모든 업무 진행 |
| **기초** | 정보화·자동화 기반이 거의 없는 단계 |
| **중간 1** | 일부 설비 자동화, 기초 시스템 도입 |
| **중간 2** | **MES·WMS 등 통합 관리 시스템 활용** |
| **고도화** | **AI·빅데이터·로봇 등 지능형 제조 구현 단계** |

**KNK 현재 위치 추정**: **중간 1 ~ 중간 2 수준** (HAIST_WORKS 자체 구축 중, baby PMS + web 통합 시스템 운영)
→ 수준확인 받으면 공식 레벨 인증 확보

#### 1-4. 전략적 가치 (팩트)

- 수준확인 받으면 → **향후 다른 스마트공장 지원사업(AI트랙 등) 자격 획득**
- AI트랙이 명시하는 "스마트공장 구축 완료 이력 (수준확인 사업 포함)" 조건 충족 가능
- 즉 **수준확인 → AI트랙** 단계적 접근 경로

#### 1-5. 신청 방법·일정

- 신청 경로: **스마트공장 사업관리시스템** (https://www.smart-factory.kr)
- 공고 상세: [bizinfo 공고 페이지](https://www.bizinfo.go.kr/web/lay1/bbs/S1T122C128/AS/74/view.do?pblancId=PBLN_000000000116004)
- 일정: 공고 원문 확인 필수

#### 1-6. 실무 확인 필요 항목

- [ ] 자체 구축 중인 HAIST_WORKS가 "자체 역량 스마트공장 구축"으로 인정되는지 확인 (전화 1357)
- [ ] "정부 스마트공장지원사업 미참여" 조건 확정 (KNK가 과거 관련 사업 수혜 없음 확인)
- [ ] 신청 기간·서류 확인

| **A조합** | ① 수준확인 (자격 만들기) + ⑤ 세액공제 (이미 진행 중 비용 회수) |
| **B조합** | ② AI 바우처 (외부 솔루션 구매) + ⑤ 세액공제 |
| **C조합** | ③ 디지털 전환 (최대 1억) + ⑤ 세액공제 |
| **D조합** | ① 수준확인 → (다음해) AI트랙 재신청 장기 플랜 + ⑤ 세액공제 즉시 |

**핵심 관찰**:
- ⑤ R&D 세액공제는 **다른 모든 대안과 병행 가능** (정부 출연금 제외 자체 부담분만)
- ① 수준확인은 **비용 전액 지원이지만 현금은 아님** — "자격 생성" 용도
- ② AI 바우처는 **공급기업 매칭 필요** — KNK 자체 구축 방향과 불일치 가능성
